## 二、设计目标

### 核心目标

**快速积累素材库，以量取胜测爆款**。用Python脚本自动化生成基础版视频，发布后通过数据筛选潜力品，再用剪映精修放大爆款。

### 成功标准

| 时间节点 | 成功标准 |
|----------|----------|
| 第8天（6/26） | 完成脚本搭建，生成首批测试视频 |
| 第12天（6/30） | 累计生成40条视频，发布20条以上 |
| 第14天（7/2） | 完成首轮数据筛选，精修5-10条 |
| 第15天+（7/3+） | 扩展到全品类，建立标准化SOP |

## 三、整体架构

### 三层流水线模型

```
第一层：批量生产层（Python脚本自动化）
    │  产出：基础版视频 × 40个（手镯+项链优先）
    ▼
第二层：数据筛选层（TikTok Analytics）
    │  筛选：按完播率、点击率、加购率排序
    ▼
第三层：精品放大层（剪映精修）
    │  产出：爆款视频 × 5-10个（二次优化发布）
    ▼
  正向循环：数据 → 优化 → 更多数据 → 更精准优化
```

### 完整工作流

```
步骤1：素材收集
  ├── 从供应商获取商品详情图（5-8张/商品）
  ├── 按商品分类存入文件夹（手镯/、项链/...）
  └── 从products.json读取商品信息（名称、价格、卖点）

步骤2：文案生成
  ├── 提取商品核心卖点（中文）
  ├── AI翻译为越南语
  ├── 生成5套不同角度的文案模板
  └── 存入文案库

步骤3：BGM准备
  ├── 从TikTok Creative Center找越南热门歌曲
  ├── 下载20首热门BGM（带鼓点的优先）
  ├── 标记每首歌的卡点节奏（BPM、鼓点位置）
  └── 存入音乐库

步骤4：批量生成视频（脚本一键跑）
  ├── 选择模板（卡点风/高级风）
  ├── 选择BGM
  ├── 自动匹配图片顺序
  ├── 自动添加越南语字幕
  ├── 自动卡点剪辑
  └── 批量导出MP4

步骤5：快速质检
  ├── 检查字幕是否正确
  ├── 检查画面是否清晰
  ├── 检查音乐卡点是否对齐
  └── 不合格的重新生成

步骤6：发布
  ├── 每天3-5条，错峰发布
  ├── 绑定对应商品小黄车
  ├── 发布时间：越南时间19:00-22:00
  └── 记录发布时间和商品对应关系

步骤7：数据追踪（每周）
  ├── 导出TikTok Analytics数据
  ├── 按完播率、点击率、加购率排序
  ├── 筛选Top20%潜力商品
  └── 进入精品放大层

步骤8：剪映精修（对潜力品）
  ├── 优化转场和特效
  ├── 调整节奏和字幕
  ├── 添加热门标签和贴纸
  └── 二次发布测数据
```

## 五、视频模板设计

### 模板A：快节奏卡点风（主力模板）

**定位**：追求完播率和曝光量，适合冷启动堆量

**参数**：
- 时长：15-20秒
- 图片数量：6-8张
- BPM要求：120-140（快节奏鼓点）

**时间轴结构**：

| 时间段 | 内容 | 画面效果 |
|--------|------|----------|
| 0-3秒 | 钩子：第1张主图 + 越南语钩子文字（大字居中） | BGM第一个重鼓点，快速推拉 |
| 3-12秒 | 卡点展示：第2-6张图（每张1-1.5秒，跟鼓点切换） | 每张配1个卖点关键词，左右滑动切换 |
| 12-15秒 | 价格+CTA：第7-8张细节图 + 价格 + 越南语CTA | BGM高潮收尾 |

**画面效果**：
- 图片切换：快速推拉/左右滑动（模拟动态感）
- 文字动画：弹跳出现/淡入淡出
- 滤镜：轻微提亮+饱和度增强

### 模板B：简约高级风（辅助模板）

**定位**：追求转化率和品牌质感，适合A/B测试对比

**参数**：
- 时长：15-20秒
- 图片数量：4-5张
- BPM要求：80-100（中慢节奏）

**时间轴结构**：

| 时间段 | 内容 | 画面效果 |
|--------|------|----------|
| 0-4秒 | 品牌开场：纯色背景 + 商品名 + 第1张图缓慢放大 | Ken Burns效果，BGM轻柔前奏 |
| 4-12秒 | 质感展示：第2-4张图（每张2-3秒，缓慢过渡） | 每张配1句描述性文案，淡入淡出 |
| 12-15秒 | 价格+CTA：第5张主图 + 价格 + "Mua ngay" | BGM渐弱收尾 |

**画面效果**：
- 图片切换：缓慢淡入淡出
- 文字动画：优雅滑入
- 滤镜：高级灰调/暖色调

## 七、BGM卡点方案

### BGM获取流程

1. 从TikTok Creative Center获取越南热门歌曲（筛选地区：Vietnam）
2. 下载20首BGM，优先选带明显鼓点的歌曲
3. 每首歌截取15-20秒高潮段落
4. 统一音量标准化（-3dB）

### 音乐信息库结构


### 卡点匹配逻辑

- **模板A**：BPM 120-140，图片切换对齐beats数组，钩子文字在第一个强鼓点出现，CTA在climax_start位置
- **模板B**：BPM 80-100，图片每2-3秒切换（不跟鼓点），文字动画跟随beats但速度减半

### BGM分配策略

| 策略 | 说明 |
|------|------|
| 轮换使用 | 20首BGM循环分配，避免同质化 |
| 品类匹配 | 手镯/项链→优雅类BGM；眼镜/耳环→活泼类BGM |
| A/B测试 | 同一商品用2种不同BGM，对比数据 |
| 定期更新 | 每周更新1次BGM库，跟随TikTok趋势 |

### 主流程伪代码

```python
def batch_generate():
    # 1. 读取商品数据
    products = load_products("products.json")
    
    # 2. 筛选优先品类（手镯+项链）
    priority_products = filter_by_category(products, ["手镯", "项链"])
    
    # 3. 加载BGM库
    bgm_library = load_bgm_library("音乐库/")
    
    # 4. 为每个商品生成视频
    for product in priority_products:
        # 4.1 选择模板（A/B轮换）
        template = select_template(product)
        
        # 4.2 选择BGM
        bgm = select_bgm(bgm_library, template)
        
        # 4.3 生成越南语文案
        subtitle = generate_subtitle(product, template)
        
        # 4.4 加载商品图片
        images = load_images(f"素材库/{product.category}/{product.id}/")
        
        # 4.5 分析BGM卡点
        beats = analyze_beats(bgm)
        
        # 4.6 合成视频
        video = compose_video(images, subtitle, bgm, beats, template)
        
        # 4.7 质检
        if quality_check(video):
            export_video(video, f"生成视频/{date}/{product.id}.mp4")
            log_success(product.id)
        else:
            log_failure(product.id, "质检失败")
```

## 九、数据筛选与精品放大

### 数据追踪指标

| 指标 | 含义 | 筛选权重 |
|------|------|----------|
| 播放量 | 视频曝光量 | 30% |
| 完播率 | 看完的比例 | 25% |
| 点击率 | 点击小黄车的比例 | 20% |
| 加购率 | 加入购物车的比例 | 15% |
| 转化率 | 最终下单的比例 | 10% |

**综合评分** = 播放量×0.3 + 完播率×0.25 + 点击率×0.2 + 加购率×0.15 + 转化率×0.1

### 筛选流程（每周一次）

```
每周日晚上：
  ├── 导出本周所有视频数据
  ├── 计算综合评分
  ├── 排序，取Top20%（约6-8个商品）
  └── 进入精品放大层

判断标准：
  ├── 潜力品：综合评分 > 50分 → 剪映精修
  ├── 平庸品：综合评分 30-50分 → 换模板/BGM重做
  └── 淘汰品：综合评分 < 30分 → 不再投入
```

### 精品放大层操作

| 优化项 | 具体操作 |
|--------|----------|
| 开场优化 | 加闪屏特效、动态文字、悬念开头 |
| 节奏调整 | 根据数据调整图片停留时间 |
| 字幕优化 | 加大字号、换颜色、加背景框 |
| BGM替换 | 换当周最热BGM，重新卡点 |
| 加贴纸 | 加价格标签、限时标签、箭头引导 |
| 加片尾 | 加品牌片尾"Follow @Cool Base for more" |

## 十、执行计划与时间线

## 十一、风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| 供应商图片质量差 | 视频效果不佳 | 要求供应商提供高清图，或自行用AI增强 |
| BGM版权问题 | 视频被下架 | 只使用TikTok平台内热门歌曲 |
| 越南语翻译不准确 | 影响转化率 | 人工抽检10%，关键文案找母语者校对 |
| 脚本运行报错 | 生产中断 | 添加异常处理和日志，单商品失败不影响整体 |
| 视频数据全零 | 无法筛选 | 跑满7天再判断，冷启动阶段数据积累需要时间 |
| TikTok限流静态图片 | 曝光低 | 模板加入动态效果（推拉、滑动），避免纯静态 |

## 十二、约束与注意事项

1. **视频时长**：15-25秒（知识库要求）
2. **完播率目标**：>40%（知识库要求）
3. **发布频率**：日更3-5条（冷启动阶段）
4. **发布时间**：越南时间19:00-22:00（知识库要求）
5. **文案语言**：越南语（面向越南市场）
6. **BGM来源**：TikTok热门越南歌曲（获取平台流量加持）
7. **视频格式**：1080×1920竖屏，MP4，30fps
8. **避免硬广**：东南亚用户反感直接推销，需软性植入
9. **避免静态图片**：东南亚市场开始限流静态图片内容，需加动态效果
