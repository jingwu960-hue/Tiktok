# TikTok 卖家大学文章抓取 & 知识库入库

> 日期：2026-06-23 | 状态：已批准

## 目标

从 `https://seller.tiktokglobalshop.com/university/` 卖家大学页面抓取所有文章（含左侧多级菜单的全部内容），经关键词 + 角色视角双层过滤后，保存到本地知识库 `/Users/wujing/QingZhen/TikTok/TikTokShop/知识库/`。

## 架构

4 阶段流水线，AI 全程主导，Playwright MCP 负责浏览器交互：

```
阶段1: 菜单提取
  Playwright 打开页面 → 提取左侧菜单完整树结构 → 生成菜单清单

阶段2: 文章抓取
  遍历菜单树 → 逐个点击菜单项 → 等待内容加载 → 提取文章文本

阶段3: 内容提炼
  第一层（关键词匹配）：按 11 个知识库类目 + 越南市场 + 时尚配件匹配
  第二层（角色视角）：按 6 个角色提取各自关心的内容，去重合并

阶段4: 保存入库
  提炼后的文章 → 保存为 .md 文件 → 更新 index.json 索引
```

## 阶段详解

### 阶段1：菜单提取

- 用 `playwright_navigate` 打开目标 URL
- 用 `playwright_evaluate` 执行 JS 提取左侧菜单 DOM，输出树形 JSON
- 包含：层级、名称、链接/文章ID、是否为外部链接
- 展示菜单清单给用户确认

### 阶段2：文章抓取

- 遍历菜单树叶子节点，点击菜单项，等待右侧内容加载
- 用 `playwright_get_visible_text` 提取文章正文
- 外部链接用 `playwright_navigate` 跳转后提取
- 每篇文章临时保存 `.raw.md`，支持断点续传

### 阶段3：内容提炼

**第一层 — 关键词匹配过滤**：
- 11 个知识库类目：选品、上架、定价、视频、直播、达人、广告、账号、冷启动、政策、问题
- 店铺相关：越南、东南亚、时尚配件、戒指、眼镜、手表、手镯、跨境、无货源
- 完全无关的文章直接丢弃

**第二层 — 角色视角提炼**：
- 运营总监：战略规划、风险管控、资源分配
- 运营教练：可执行步骤、时间节点、冷启动方法
- 选品专家：选品方法论、品类分析、利润计算
- 运营专家：上架技巧、标题公式、SEO、卖点提炼
- 数据分析师：指标定义、数据解读、优化策略
- 脚本编剧：可用于视频脚本的素材、故事、案例

### 阶段4：保存入库

- 文件命名：`卖家大学-{分类}-{文章标题}.md`
- 文件头部元信息：
  ```markdown
  <!-- source: {原始URL} -->
  <!-- categories: {匹配的知识库类目} -->
  <!-- roles: {关联的角色} -->
  <!-- crawled_at: {抓取时间} -->
  ```
- 更新 `index.json` 的 `entries` 数组
- 生成抓取报告

## 技术选型

| 项目 | 选择 |
|------|------|
| 抓取工具 | Playwright MCP |
| 菜单提取 | `playwright_evaluate` + JS DOM 遍历 |
| 内容提炼 | AI 直接处理 |
| 保存格式 | Markdown + HTML 注释元信息 |

## 风险

| 风险 | 应对 |
|------|------|
| 菜单在 iframe 内 | 使用 `playwright_iframe_*` 系列工具 |
| 外部链接 | 跳转后同方式提取 |
| 页面加载慢 | 点击后等待 2-3 秒 |
| 文章数量 >50 | 分批处理，每批 10 篇暂停汇报 |
