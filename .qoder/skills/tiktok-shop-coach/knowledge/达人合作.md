# 达人合作指南

## 达人合作价值

- **快速曝光**：借助达人粉丝基础
- **信任背书**：粉丝对达人推荐更信任
- **内容质量**：专业团队制作的视频质量更高
- **降低风险**：不依赖单一渠道

---

## 达人选择标准

### 基础指标

| 指标 | 要求 | 说明 |
|------|------|------|
| 粉丝数 | 1万+ | 越南达人门槛 |
| 平均播放 | 5000+ | 验证账号真实性 |
| 互动率 | >3% | 粉丝活跃度 |
| 带货视频占比 | >30% | 专业度指标 |

### 进阶指标

- 账号内容垂直度
- 粉丝画像匹配度
- 往期带货数据
- 评论区粉丝反馈

---

## 达人合作模式

### 纯佣模式（新手推荐）

- **定义**：按实际销售额支付佣金，不支付坑位费
- **佣金比例**：15-30%
- **优势**：零风险，按效果付费
- **适合**：刚起步，预算有限

### 坑位费+佣金模式

- **定义**：支付固定坑位费 + 按销售额支付佣金
- **坑位费**：100-500元不等
- **佣金比例**：10-20%
- **优势**：达人更愿意用心制作
- **适合**：有一定预算，追求质量

### 买断模式

- **定义**：一次性支付固定费用，不与销售额挂钩
- **费用**：500-2000元
- **优势**：内容版权归商家所有
- **适合**：品牌方，长期复用

## 合作流程

### Step 1：筛选达人

1. 使用Kalodata/EchoTik筛选越南达人
2. 查看达人历史带货数据
3. 分析粉丝画像（年龄、性别、地区）
4. 评估内容质量

### Step 2：联系达人

**联系模板**：
```
您好！

我是越南TikTok Shop卖家，销售[产品类目]。

我们正在寻找优质达人合作。

[达人名字]，您的内容风格和我们产品很匹配，想了解合作意向。

请问：
1. 是否接受纯佣合作？
2. 或者有其他合作方式？
3. 您的联系方式是？

期待合作！
```

### Step 3：寄样评估

1. 确认合作意向后，寄送样品
2. 样品数量：根据视频数量决定
3. 附上产品卖点文档
4. 约定视频交付时间

### Step 4：内容审核

1. 达人提交视频预览
2. 审核内容是否符合要求
3. 提供修改意见
4. 确认后发布

### Step 5：数据追踪

1. 生成达人专属链接
2. 追踪视频播放和转化数据
3. 分析ROI，决定是否继续合作
4. 记录达人表现

## 寄样模板

### 寄样前准备

- 产品选择：选择有爆款潜力的品
- 样品数量：每条视频1-3件
- 附赠物料：产品卖点卡、注意事项

### 物流选择

| 方式 | 时效 | 费用 | 推荐 |
|------|------|------|------|
| 空运 | 3-5天 | 高 | 紧急样品 |
| 陆运 | 7-10天 | 中 | 常规样品 |
| 海运 | 15-20天 | 低 | 大批量 |

## 注意事项

### ❌ 不要做

- 不要同时合作太多达人（管理不过来）
- 不要不给样品就让达人拍视频
- 不要只看粉丝数不看转化数据
- 不要不给达人明确的视频要求

### ✅ 正确做法

- 先从纯佣开始，降低风险
- 找数据真实的达人（查Kalodata验证）
- 给达人提供详细的产品卖点
- 建立长期合作关系
- 记录每个达人的ROI表现
