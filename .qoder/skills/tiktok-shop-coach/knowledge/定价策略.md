# 定价策略

## 定价公式

```
售价 = 采购价 + 跨境运费 + 平台佣金 + 利润
```

---

## 成本参数详解

### 采购价

- 来源：1688/拼多多
- 目标：≤20元
- 谈判技巧：多比较3家供应商，争取批量优惠

### 跨境运费

| 重量区间 | 运费（元/kg） | 说明 |
|----------|---------------|------|
| <100g | 2.3 | 轻小件优先 |
| 100-500g | 2.5-2.8 | 注意体积重 |
| 500g-1kg | 2.8-3.0 | 临界区间 |
| >1kg | 3.0+ | 超重加价 |

### 平台佣金

| 市场 | 佣金率 | 说明 |
|------|--------|------|
| 越南 | 6% | 基础佣金 |
| 泰国 | 6% | 基础佣金 |
| 菲律宾 | 5% | 较低 |
| 马来西亚 | 6% | 基础佣金 |

### 交易手续费

- 2026年新规：6%
- 部分类目可能有额外费用

## 定价示例

### 示例1：太阳镜

| 成本项 | 金额 |
|--------|------|
| 采购价 | 10元 |
| 国内运费 | 2元（到货代） |
| 货代费 | 1.5元 |
| 跨境运费 | 2元（0.2kg） |
| 平台佣金 | 售价×6% |
| **总成本** | **15.5元 + 佣金** |

**定价方案**：
- 保守定价：28元 → 佣金1.68元 → 利润10.82元（利润率38.6%）
- 标准定价：32元 → 佣金1.92元 → 利润14.58元（利润率45.6%）
- 激进定价：36元 → 佣金2.16元 → 利润18.34元（利润率50.9%）

## 定价技巧

### 1. 越南盾显示规则

- 金额四舍五入到千位
- 使用三位分隔符（如 174,240 VND）
- 标注汇率更新时间

### 2. 价格锚点设置

- 设置原价和促销价两个价格
- 促销价必须是原价的70-80%
- 营造"捡到便宜"的感觉

### 3. 心理定价法

| 策略 | 例子 | 效果 |
|------|------|------|
| 尾数定价 | 99,000 VND | 感觉不到100 |
| 套餐定价 | 3件199元 | 提高客单价 |
| 梯度定价 | S/M/L三个价 | 满足不同需求 |

## 定价避坑

### ❌ 定价过低的后果

- 利润太薄，扛不住退货
- 被认为是假货或劣质品
- 引发价格战，破坏市场

### ❌ 定价过高的后果

- 转化率断崖式下跌
- 失去自然流量
- 用户流失到竞争对手

### ✅ 合理定价区间

- 新品期：利润率可以稍低（20-25%）
- 成长期：恢复到目标利润（30%+）
- 爆款期：可以适当提价
