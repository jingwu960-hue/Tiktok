# TRAE IDE → TRAE Work 迁移操作指南

## 已自动完成的步骤

以下步骤已由 AI 助手自动完成，无需手动操作：

### 1. Skill 打包
- `tiktok-shop-coach.zip` — TikTok Shop 运营助手技能包
- `ecom-image-translate.zip` — 电商图片翻译重绘技能包

### 2. AGENTS.md 创建
- 已在项目根目录创建 `AGENTS.md`，包含项目背景、关键信息、数据文件路径和行为指引

---

## 需要手动完成的步骤

以下步骤需要在 TRAE Work 中手动操作：

### 步骤 A：导入 Skill（如使用 TRAE Work 网页版）

> 如果使用 TRAE Work **桌面版**打开同一项目目录，Skill 会自动识别，可跳过此步骤。

1. 打开 TRAE Work 网页版（https://work.trae.cn）
2. 进入 **技能管理中心**
3. 点击 **上传技能**
4. 分别上传以下两个 zip 文件：
   - `tiktok-shop-coach.zip`
   - `ecom-image-translate.zip`
5. 确认技能安装成功

### 步骤 B：创建全局规则

1. 打开 TRAE Work → **设置** → **规则** → **创建规则**
2. 输入规则内容：`ai返回内容用中文展示`
3. 生效方式选择：**始终生效**
4. 保存

### 步骤 C：开启 AGENTS.md 上下文

> 仅 TRAE Work 桌面版支持此功能

1. 打开 TRAE Work → **设置** → **规则** → **导入设置**
2. 开启 **"将 AGENTS.md 包含在上下文中"**
3. 保存

### 步骤 D：配置 MCP Server（如需要）

1. 打开 TRAE Work → **设置** → **MCP Server** → **添加**
2. 搜索并添加 **Playwright** MCP Server
3. 注意：浏览器工具（integrated_browser）为内置功能，无需额外配置

### 步骤 E：创建定时自动化任务（可选）

TRAE Work 支持定时自动化任务，这是 TRAE IDE 不具备的新能力。如需创建：

1. 打开 TRAE Work → 左栏顶部 **自动化**
2. 点击 **在对话中创建** 或 **手动新建**
3. 配置任务名称、触发时间、任务内容
4. 选择运行模式（Work/Code）、运行环境（云端/本地）
5. 选择输出内容存储位置
6. 点击 **创建**

## 功能差异备忘

| 功能 | TRAE IDE | TRAE Work | 说明 |
|------|----------|-----------|------|
| **Hook 自动化** | 支持 | 不支持 | TRAE Work 无 Hook 机制，可用 Skill+Rule 替代 |
| **Memory 记忆** | 支持 | 不支持 | 需将重要记忆转为全局规则或 AGENTS.md |
| **项目规则文件** | `.trae/rules/` | 不支持 | 用 AGENTS.md 替代（仅桌面版） |
| **自定义智能体** | 支持 | 不支持 | 用 Skill + Rule 组合替代 |
| **定时自动化** | 不支持 | 原生支持 | TRAE Work 独有优势 |
| **三端同步** | 不支持 | 支持 | 网页版/桌面版/移动版任务同步 |
| **飞书集成** | 不支持 | 支持 | 可用于通知推送 |
