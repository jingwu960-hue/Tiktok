## 二、综合成本计算

### 完整成本公式

```
总成本 = 采购成本 + 平台费用 + 达人佣金 + 广告费用 + 退货损耗 + 汇率损耗
```

### 示例计算（售价60元）

假设：
- 采购成本：20元
- 售价：60元（约198,000 VND）
- 达人佣金：20%
- 广告费用：10%

| 成本项 | 计算 | 金额 |
|--------|------|------|
| 采购成本 | - | 20元 |
| 平台佣金 | 60 × 15.5% | 9.3元 |
| 交易手续费 | 60 × 6% | 3.6元 |
| 达人佣金 | 60 × 20% | 12元 |
| 广告费用 | 60 × 10% | 6元 |
| 退货损耗 | 60 × 5% | 3元 |
| 汇率损耗 | 60 × 2% | 1.2元 |
| **总成本** | - | **55.1元** |
| **净利润** | 60 - 55.1 | **4.9元** |
| **净利润率** | 4.9 ÷ 60 | **8.2%** ⚠️ |

> ⚠️ **警示**：如果计入所有成本，净利润率仅8.2%，风险极高！

## 三、合理定价策略

### 1. 定价公式（全成本考量）

```
售价 = 采购成本 ÷ (1 - 平台费率 - 达人佣金 - 广告费用 - 退货损耗 - 汇率损耗 - 目标利润率)
```

### 2. 定价计算示例

假设：
- 采购成本：20元
- 平台费率：21.5%
- 达人佣金：20%
- 广告费用：10%
- 退货损耗：5%
- 汇率损耗：2%
- 目标利润率：25%

**计算**：
```
售价 = 20 ÷ (1 - 0.215 - 0.20 - 0.10 - 0.05 - 0.02 - 0.25)
     = 20 ÷ 0.165
     = 121元（约400,000 VND）
```

### 3. 简化定价公式

**推荐公式**：`售价 = 采购价 × 4 ~ 5`

**原因**：
- 需要覆盖平台费用21.5%
- 需要覆盖达人佣金15-20%
- 需要覆盖广告费用10%
- 需要预留利润空间20-25%

## 九、风险警示

### ❌ 不要做的事

1. **不要用"采购价×2.8"粗放定价**
   - 忽略了达人佣金和广告费用
   - 净利润率可能为负

2. **不要忽视达人佣金**
   - 达人佣金15-30%，是最大的隐性成本
   - 必须在选品阶段就纳入计算

3. **不要盲目参与达人合作**
   - 先用自建内容测试市场
   - 确认利润空间后再合作

### ✅ 必须做的事

1. **必须用全成本公式重新定价**
   - 售价 = 采购价 × 4~5

2. **优先自建内容获得免费流量**
   - 降低对达人和广告的依赖

3. **积极参与平台营销活动获得补贴**
   - Smart Promotion、平台大促等

## 十、总结

### 核心结论

1. **定价必须考虑全成本**：平台费用21.5% + 达人佣金20% + 广告10% = 51.5%
2. **定价倍率建议**：采购价 × 4~5
3. **利润率目标**：净利润率20-25%
4. **优先策略**：自建内容 > 参与平台活动 > 达人合作 > 广告投放

### 成本控制优先级

| 优先级 | 策略 | 效果 |
|--------|------|------|
| 🔴最高 | 提升定价倍率（×4~5） | 确保利润空间 |
| 🔴高 | 自建内容，减少达人依赖 | 节省15-30% |
| 🟡中 | 参与Smart Promotion | 获得平台补贴 |
| 🟡中 | 优化视频质量，获得免费流量 | 降低广告成本 |
| 🟢低 | 精准投放广告 | 提升ROI |
