# TikTok带货视频生成工具 - 使用说明

## 快速开始

### 1. 环境准备

```bash
# 安装系统依赖
brew install ffmpeg

# 安装Python依赖
cd TikTokShop/视频制作/脚本
pip3 install -r requirements.txt
```

### 2. 准备素材

#### 商品图片
将每个商品的详情图放入对应目录：
```
素材库/
├── 手镯/
│   ├── 商品ID1/
│   │   ├── 1.jpg
│   │   ├── 2.jpg
│   │   └── ...（5-8张）
│   └── 商品ID2/
├── 项链/
└── ...
```

#### BGM音乐
将越南热门歌曲MP3放入：
```
音乐库/
├── bgm01.mp3
├── bgm02.mp3
└── ...（20首）
```

### 3. 生成视频

```bash
cd TikTokShop/视频制作/脚本
python3 main.py
```

视频将输出到：`生成视频/YYYY-MM-DD/商品ID.mp4`

### 4. 质检
脚本自动质检，不合格的视频会跳过并记录日志。

### 5. 发布
- 每天3-5条
- 发布时间：越南时间19:00-22:00（北京时间20:00-23:00）
- 绑定对应商品小黄车
- 发布后在`已发布/发布记录.json`中记录

| 目录 | 用途 |
|------|------|

## 模板说明

### 模板A：快节奏卡点风
- 时长：15秒
- 图片：6张
- BGM：120-140 BPM
- 适用：冷启动堆量

### 模板B：简约高级风（待实现）
- 时长：18秒
- 图片：4张
- BGM：80-100 BPM
- 适用：转化率优化

## 文案模板

| 模板 | 适用场景 | 钩子文案 |
|------|----------|---------|
| 痛点型 | 手镯/项链 | Đừng đeo trang sức rẻ tiền nữa! |
| 颜值型 | 眼镜/耳环 | Xinh hơn chỉ với 1 món phụ kiện |
| 情感型 | 情侣戒 | Quà tặng ý nghĩa cho người yêu |
| 性价比型 | 全品类 | Đẹp mà rẻ, ai cũng muốn có |
| 潮流型 | 全品类 | Xu hướng 2026 hot nhất |

## 数据筛选流程

1. 每周日导出TikTok Analytics数据
2. 计算综合评分（播放量30% + 完播率25% + 点击率20% + 加购率15% + 转化率10%）
3. 筛选Top20%潜力商品
4. 用剪映精修潜力视频，二次发布

## 技术架构

```
Python脚本批量生成 → TikTok发布 → 数据筛选 → 剪映精修
```

### 核心模块

| 模块 | 文件 | 功能 |
|------|------|------|
| 文件管理 | utils/file_manager.py | 商品读取、品类筛选、图片加载 |
| 越南语翻译 | utils/translator.py | 5套文案模板、AI翻译 |
| 图片处理 | modules/image_processor.py | 裁剪、滤镜、文字叠加、动画 |
| BGM分析 | modules/bgm_analyzer.py | 鼓点检测、卡点匹配 |
| 视频合成 | templates/template_a_fast.py | 模板A视频生成 |
| 质检 | modules/quality_check.py | 视频质量检查、发布记录 |
| 主入口 | main.py | 批量生成视频 |
