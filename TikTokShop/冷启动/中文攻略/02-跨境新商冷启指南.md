# 【美国】跨境新商冷启指南

> TikTok Shop经营方法论｜新商成长"三步走"V1.1
> 更新时间：2026-06-24

## 一、大量商家已在TikTok Shop上抓住出海新机遇

TikTok Shop是一站式电商解决方案，助力商家在TikTok上通过短视频、直播、商城等多场域协同互通，实现出海生意的新增长。自TikTok Shop开启美区业务以来，越来越多的商家在"内容+货架"双驱动电商模式下，通过强大的选品能力、内容运营和品牌营销能力，实现了生意的高速增长：

- 某女装商家：入驻20天日均GMV超$5千，订单量增长2000%
- 某数码商家：入驻30天日均GMV超$3千，现累计GMV已超～$150万
- 某箱包商家：入驻40天GMV超$10万，现已成为美区箱包类目Top5商家

为了帮助新商家更好地履约订单，所有新商家在入驻TikTok Shop后都将进入新手村即「新店铺考察期」，熟悉平台履约和服务规则，提升在TikTok Shop上的运营效率。本期将重点讲述新商家从0到1冷启方法论，帮助商家快速出村放大生意规模。

## 二、新商家成长"三步走"，快速出村放大生意规模

首先，新入驻商家需在商家后台里了解「新店铺考察期」的指标要求，熟悉TikTok Shop的运营规则，经营店铺的同时做好商品体验、物流履约体验、客户服务体验。

### 新手村出村规则（2024/07/10正式更新）

| 考察要求 | 商家中心查看路径 |
|---|---|
| 累计订单量>=500 | 商家中心首页 |
| 累计在村时长>=30天 | 查看店铺考察期表现 |
| 商责店铺差评率<0.4% | 熟悉新手村6大考察要求 |
| 延迟履约率<4% | 查看任务中心完成新手任务 |
| 商责取消率<2.5% | 帮助中心提交工单 |
| 违规分<12 | 咨询在线客服 |

**操作路径**：商家中心首页 → 完成必做任务 → 查看店铺考察期表现 → 熟悉6大考察要求 → 完成新手任务 → 寻求平台帮助

在提升买家购物体验的基础上，基于过往商家在美区早期经营的成功经验，总结出三条有效的冷启起量路径：

- **第一步**：做好选品，聚焦商家独立站/美区电商平台的爆品，快速上架并积极与达人寄样测品
- **第二步**：做好短视频供给，其中合作达人数和短视频投稿量决定了生意的量变到质变
- **第三步**：做好短视频投流，放大动销带货短视频的转化效率，持续推爆品

做好运营"三步走"之外，同时要重点关注店铺体验分，实时关注店铺运营效率和健康度，改善买家购物体验，提升买家转化率。

### 第一步：做好选品（1-7天）

- **关键动作**：参考TikTok Shop站内爆款/商家独立站/美区电商平台的爆款迅速上架商品，选出3-5个主推款与联盟达人合作
- **目标指标**：上架商品～10-20个，联盟达人主推带货商品～3-5个
- **操作路径**：「商家后台」-「管理全球商品」-添加选品

### 第二步：做好短视频供给（7-30天）

- **关键动作**：保证带货短视频投稿量15+，优先建联联盟达人、投稿带货短视频，同时也要做好商家自营短视频

**找什么样的达人？**
- 保持「原生感」，选择美国当地达人，内容创作上贴近美区用户的语言方式和生活习惯
- 主要考察达人7-14天的短视频播放量、发帖勤奋度（7天内发布3～4条短视频）
- 粉丝量不是最关键的

**找多少个联盟达人？**
- 日均建联达人100+
- 周均合作达人15+

**投稿什么样的短视频？**
- 保持原生感，时长控制在～30秒，最长不超过1.5分钟
- 讲清楚商品的3个主要卖点
- 明确指引用户点击挂车商品链接，下单购买

**投稿多少个短视频?**
- 月均带货短视频20+
- 达人短视频 : 商家自卖短视频稿比例～4：1

**商家怎么找联盟达人合作**：
操作路径：「商家后台」-「联盟带货」-创建联盟计划
- 创建「公开合作」计划，所有达人均可见
- 创建「定向合作」计划，邀请特定达人合作

**达人建联话术**：用词尽量口语化，不要过于正式，简要介绍：
- 我是谁(品牌)
- 我卖什么的(产品)
- 我的佣金多少（主推产品佣金率～10-20%）
- 我产品厉害在哪儿(核心卖点2-3点)
- 你可以怎么展示(表达形式)

### 第三步：做好短视频投流（31-60天）

- **关键动作**：重点投流动销短视频素材，稳定流量获取，持续推爆品，大促期间加大广告投入
- **目标指标**：广告消耗$600+，广告ROAS（即广告GMV/广告支出）：～2-3，广告GMV占总GMV～20-30%
- **操作路径**：「商家后台」-「店铺广告」中联系销售团队或发邮件开通广告账号，进行投流

### 经营基础：做好店铺体验分（全经营阶段）

「店铺体验分」是基于「商品满意度」、「履约物流体验分」、「客户服务体验分」综合计算得出，并对所有同类商家进行排名。商家要基于以上三个维度为用户提供优质的购物体验。

*店铺评分仅适用于在过去90个日历日内至少完成30个订单且无虚假订单或风险订单的卖家*

若某店铺体验分为4分（满分5分），意味着该店铺处于第80百分位，有20%的店铺优于该店铺。

**评估指标**：
- 商品满意度：差评率（NRR）、非买家责任退货退款（NBFR）
- 履约物流体验分：商责取消率（SFCR）、延迟履约率（LDR）
- 客户服务体验分：买家投诉率（CCR）、24小时回复率

## 三、新商家经营最佳实践

### 案例1: 某休闲女装商家

**【商家背景】**
某美区头部电商平台休闲女装商家，23年底入驻TikTok Shop，上架商品14+，客单价$20-30

**【冷启亮点】**
- 入驻20天后日均GMV迅速超$5千
- 以短视频出单为主（短视频GMV占比90%+，主要由达人带货短视频贡献）

**【关键打法】**

| 步骤 | 关键动作及指标 |
|---|---|
| 第一步：做好选品 | 开店后迅速上品，聚焦"少而精"的潜力爆品，不广撒网与达人寄样测品。上架商品14个，客单价在$20-30，联盟达人主推带货商品1-3个。冷启阶段跑出Top级爆品，单品GMV占比超90% |
| 第二步：做好短视频供给 | 前期重点通过在商家后台创建联盟计划，积极与达人建联、寄样测品，投稿带货短视频。冷启阶段：90%短视频GMV由联盟达人贡献，累计建联达人～1000+，累计合作达人～150+，累计短视频投稿～100+ |
| 第三步：做好短视频投流 | 在单条短视频起量后，开始投广告放大动销短视频素材。重点投商品点击-转化率（C_O）>2%的带货短视频，广告消耗日均500+，大促期间加大广告投入，广告ROAS～3 |

**Top达人带货短视频关键要点**：
- 本地达人的语言习惯和内容创作贴近美区本地文化，具有原生感
- 与粉丝评论互动，增强短视频代入感
- 展示试穿效果，并明确讲述商品的3个主要卖点
- 引导粉丝去短视频挂车商品链接里下单购买

**团队配置情况**：
- 2位店铺运营：主要负责店铺管理、商品上架、履约售后
- 4位达人BD：主要负责达人建联、短视频审核
- 1位广告投手：主要负责广告账号、广告投放等操作，稳定广告ROAS
