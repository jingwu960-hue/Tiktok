## 六、实战知识库

### 1. 选品策略

#### 核心原则
- **轻小件优先**：重量<200g，体积<20×15×10cm
- **客单价锚定**：$5-$15黄金区间
- **复购率思维**：美妆工具、手机配件、家居小物

#### 选品漏斗模型
1. **数据筛选**：近30天销量增长率>50%，店铺数<100，评价数>1000
2. **供应链验证**：起订量<500件，账期30天，有现货
3. **利润测算**：毛利>30%，广告预算预留20%

#### 2026年潜力品类
1. 便携式小风扇
2. 防蚊手环
3. 手机防水袋
4. 折叠收纳盒
5. 防晒冰袖

### 2. 商品上架

#### 6步上架流程
1. **整理高清主图**：800×800像素以上，纯白背景
2. **填写规格参数**：材质、尺寸、重量、颜色
3. **套用标题公式**：关键词+属性+功能+场景
4. **设置售价利润**：售价=拿货价+运费+佣金+利润
5. **选择发货模式**：新手选平台中转仓发货
6. **绑定库存发布**：库存填999，精准确认类目

#### 标题公式
```
核心关键词 + 产品属性 + 功能特点 + 适用场景
```

示例：
- 收纳盒：透明塑料收纳盒 大号加厚防尘 厨房卧室整理箱
- 太阳镜：复古椭圆金属太阳镜 防紫外线 时尚百搭

### 3. 短视频制作

#### 爆款视频结构（15秒）
1. **0-3秒**：痛点钩子 - 直击用户高频需求
2. **3-8秒**：解法演示 - 展示产品效果
3. **8-15秒**：行动指令 - 引导购买

#### 三大爆款模板
1. **前后对比法**：展示使用前后强烈反差
2. **场景痛点法**：还原日常生活槽点
3. **反常识惊喜法**：超出预期的功能

#### 拍摄技巧
- **设备**：手机+环形灯+稳定器
- **音乐**：东南亚热门TikTok神曲
- **演员**：优先本地模特
- **时长**：15-25秒最佳

#### 内容矩阵
- 60%种草视频：展示痛点解决
- 30%品牌视频：知识教程分享
- 10%互动视频：蹭热点提活跃

### 4. 冷启动策略

#### 60天破零计划
| 阶段 | 目标 | 核心动作 |
|------|------|----------|
| 第1-15天 | 打底测款 | 日更3-5条视频，批量寄样100份 |
| 第16-30天 | 投流实测 | 日投25美金，筛选高效素材 |
| 第31-45天 | 稳定出单 | 日更10条，沉淀200+素材 |
| 第46-60天 | 模型固化 | 锁定优质素材，形成标准化流程 |

#### 四大出单渠道
1. **商品卡**：日常优化+小额投流
2. **自营短视频**：日更3-10条
3. **达人合作**：批量寄样+筛选优质
4. **直播**：短时高频（30-60分钟/次）

### 5. 定价策略

#### 全成本公式
```
售价 = 采购成本 ÷ (1 - 平台费率 - 达人佣金 - 广告费用 - 退货损耗 - 汇率损耗 - 目标利润率)
```

#### 简化公式
```
售价 = 采购价 × 4 ~ 5
```

#### 成本构成（2026年）
- 平台佣金：15.5%
- 交易手续费：6%
- 达人佣金：18-22%
- 广告费用：10%
- 退货损耗：5-10%
- 汇率损耗：2-3%
