## 一、6步上架流程（5分钟搞定）

### 第一步：整理高清主图（1分钟/张）

**要求**：
- **尺寸**：800×800像素以上，正方形，1200×1200最佳
- **背景**：纯白（RGB 255,255,255），无水印、无文字、无杂物
- **占比**：产品占画面80%以上，清晰展示全貌
- **数量**：1张主图+4张辅图（场景图、细节图、尺寸图、使用图）

**快速方法**：
1. 1688/拼多多找同款高清图
2. 用美图秀秀"抠图→白底→裁剪成正方形"
3. 辅图直接用厂家原图，选清晰、无中文的

### 第二步：填写规格参数（30秒）

**必填项**：
- **基础信息**：材质、尺寸、重量、颜色
- **功能特点**：核心功能、适用场景、优点
- **包装清单**：包含配件、数量
- **注意事项**：使用禁忌、保养方法

**提示**：参数不用写英文，中文填写即可，平台自动翻译。

### 第三步：套用标题公式（10秒写完）

**公式**：
```
核心关键词 + 产品属性 + 功能特点 + 适用场景
```

**示例**：
- 太阳镜：复古椭圆金属太阳镜 防紫外线 时尚百搭
- 戒指：六字真言旋转男戒 时尚个性 送礼佳品
- 手表：米兰网带女表 简约气质 送礼首选

### 第四步：设置售价利润

**基础公式**：
```
售价 = 拿货价 + 跨境运费 + 平台佣金 + 利润
```

**2026东南亚通用定价公式**：
- 拿货价：1688/拼多多价格
- 跨境运费：1kg内2.3-3元（东南亚专线）
- 平台佣金：5%-8%（TikTok Shop）
- 利润：售价的20%-30%（新手建议25%）

**示例**（拿货10元，重0.5kg）：
- 运费：2.5元
- 佣金：售价×6%
- 设定售价：25元
- 佣金：1.5元
- 利润：25-10-2.5-1.5=11元，**利润率44%**

**避坑**：
- ❌ 不要定价过低，低于20%利润容易亏本
- ❌ 不要过高，比同行高10%以上没流量

### 第五步：选择发货模式（新手必选）

**推荐**：平台中转仓发货（无货源模式）

**流程**：
买家下单 → 你去1688拿货 → 发平台中转仓 → 平台跨境运输 → 清关 → 当地派送

**优势**：
- 零库存、零风险
- 不用囤货、不用管物流

**禁选**：自发货（新手没物流渠道，丢件延误必罚款）

### 第六步：绑定库存发布（10秒完成）

- **库存**：新手填999（无货源，卖多少拿多少）
- **类目**：精准选择（如太阳镜选"时尚配件>眼镜>太阳镜"）
- **发布**：点击"立即上架"，审核10-30分钟

### 标题公式详解

```
[核心关键词] + [产品属性] + [功能特点] + [适用场景]
```

## 四、定价策略

### 成本构成

| 成本项 | 说明 | 示例 |
|--------|------|------|
| 采购成本 | 1688拿货价 | 10元 |
| 国内运费 | 到货代/中转仓 | 3元 |
| 跨境运费 | 平台补贴或自付 | 2.5元 |
| 平台佣金 | 5%-8% | 售价×6% |
| 其他费用 | 包装、退货等 | 2元 |

### 定价技巧

| 技巧 | 示例 | 效果 |
|------|------|------|
| 尾数定价 | 199,000 VND 比 200,000 VND | 提升转化率5-10% |
| 整数定价 | 200,000 VND | 提升品牌感 |
| 对比定价 | 原价250,000 → 现价199,000 | 突出优惠感 |
