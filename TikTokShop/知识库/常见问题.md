## 一、账户相关

### Q1：如何注册TikTok账号？

**A**：
1. 下载TikTok应用
2. 使用手机号或邮箱注册
3. 设置用户名和密码
4. 完善个人资料（头像、简介）
5. 开始养号操作

### Q2：TikTok账号被封怎么办？

**A**：
1. 查看封号原因通知
2. 根据提示提交申诉
3. 提供相关证明材料
4. 等待审核结果（通常3-5个工作日）

### Q3：如何绑定TikTok Shop店铺？

**A**：
1. 进入TikTok Shop卖家中心
2. 找到"绑定TikTok账号"选项
3. 输入TikTok账号信息
4. 完成绑定验证

## 二、商品上架

### Q1：商品审核需要多长时间？

**A**：通常10-30分钟，高峰期可能需要1-2小时。

### Q2：商品审核不通过怎么办？

**A**：
1. 查看审核失败原因
2. 根据提示修改商品信息
3. 重新提交审核
4. 如仍不通过，联系客服咨询

### Q3：如何批量上架商品？

**A**：
1. 使用TikTok Shop商家后台的批量上传功能
2. 下载商品模板Excel文件
3. 填写商品信息
4. 上传文件完成批量上架

## 三、物流发货

### Q1：如何选择物流方式？

**A**：新手建议选择平台中转仓发货（无货源模式），优势：
- 零库存、零风险
- 平台补贴跨境运费
- 无需自己处理清关

### Q2：发货时效有什么要求？

**A**：标准订单需在2个工作日内进入"运输中"状态，预售订单按页面标注时间发货。

### Q3：买家退货怎么办？

**A**：
1. 查看退货原因
2. 根据平台规则处理
3. 如需退款，及时操作
4. 记录退货数据，优化商品

## 四、营销推广

### Q1：如何联系达人合作？

**A**：
1. 进入TikTok Shop达人广场
2. 筛选符合品类的达人
3. 发送合作邀请
4. 协商佣金比例和合作方式

### Q2：如何投放广告？

**A**：
1. 进入TikTok Shop广告后台
2. 设置广告预算和目标
3. 选择投放方式（CPM/CPC）
4. 上传广告素材
5. 监控广告效果

### Q3：如何参加平台活动？

**A**：
1. 关注卖家中心活动页面
2. 查看活动要求和时间
3. 报名参加符合条件的活动
4. 准备活动商品和素材

## 五、账户资金

### Q1：如何设置收款账户？

**A**：
1. 进入卖家中心资金管理
2. 添加收款账户
3. 选择支持的支付方式
4. 完成账户验证

### Q2：结算周期是多久？

**A**：通常T+7结算，即订单完成后7天可提现。

### Q3：如何申请提现？

**A**：
1. 进入资金管理页面
2. 查看可提现金额
3. 发起提现申请
4. 等待到账（通常1-3个工作日）

## 六、常见错误处理

### Q1：订单取消率过高怎么办？

**A**：
1. 优化库存管理
2. 提高发货效率
3. 及时更新商品库存
4. 减少因缺货导致的取消

### Q2：差评率过高怎么办？

**A**：
1. 优化商品描述和图片
2. 提升商品质量
3. 改善包装
4. 及时处理客户问题

### Q3：店铺评分过低怎么办？

**A**：
1. 分析评分构成
2. 针对性优化
3. 提供优质客户服务
4. 定期监控评分变化

## 七、合规经营

### Q1：如何避免侵权？

**A**：
1. 不使用品牌关键词
2. 商品图片无Logo
3. 不销售仿牌商品
4. 使用商标查询工具检查

### Q2：如何处理违规通知？

**A**：
1. 查看违规原因
2. 及时整改
3. 如误判，提交申诉
4. 记录经验，避免再次违规

### Q3：如何关注平台政策更新？

**A**：
1. 关注卖家中心公告
2. 订阅平台邮件通知
3. 参加平台培训课程
4. 加入卖家交流群

## 八、视频制作相关

### Q1：视频完播率低怎么办？

**A**：
1. 优化前3秒钩子，设置悬念或痛点
2. 控制视频时长在15-25秒
3. 节奏紧凑，避免拖沓
4. 结尾设置反转或惊喜

### Q2：如何提高视频互动率？

**A**：
1. 在视频中提问引导评论
2. 设置投票或选择题
3. 回复每条评论
4. 使用热门音乐和话题标签

### Q3：视频审核不通过怎么办？

**A**：
1. 查看审核失败原因
2. 检查是否涉及违规内容
3. 修改后重新提交
4. 如仍不通过，联系客服咨询

## 九、直播带货相关

### Q1：直播没人看怎么办？

**A**：
1. 优化直播标题和封面
2. 选择合适的开播时间
3. 发福袋吸引人气
4. 邀请达人或朋友助力

### Q2：直播转化率低怎么办？

**A**：
1. 优化商品讲解话术
2. 设置限时优惠制造紧迫感
3. 展示商品真实使用效果
4. 及时回复弹幕互动

### Q3：直播设备推荐？

**A**：
1. 手机：iPhone 12+或安卓旗舰
2. 补光：18寸环形灯
3. 收音：领夹麦或无线麦
4. 支架：多角度手机支架

2. 关注市场趋势
3. 测试不同品类，找到爆款
