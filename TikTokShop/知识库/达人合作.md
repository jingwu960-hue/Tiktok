## 一、达人合作概述

### 1.1 为什么达人合作重要

达人合作是TikTok Shop东南亚最核心的出单渠道之一。相比于商家自营短视频，达人带货具有以下优势：

| 优势 | 说明 |
|------|------|
| **信任度高** | 粉丝信任达人推荐，转化率更高 |
| **流量大** | 达人粉丝基础带来海量曝光 |
| **省时省力** | 不需要商家自己制作视频 |
| **快速起量** | 达人视频容易出爆款 |
| **本地化** | 达人更了解本地文化和审美 |

### 1.2 达人合作核心指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| **带货视频数** | 达人发布的带货视频数量 | 目标≥50条 |
| **GMV贡献** | 达人带来的销售额占比 | 目标30-50% |
| **平均ROI** | 投入产出比 | 目标≥1:3 |
| **合作达人数** | 稳定合作的达人数量 | 目标10-20个 |

## 二、达人分类与选择标准

### 2.1 达人等级划分

| 等级 | 粉丝数 | 带货能力 | 适合阶段 | 合作成本 |
|------|--------|----------|----------|----------|
| **头部达人** | 100万+ | 强 | 品牌推广 | 高（坑位费+佣金） |
| **腰部达人** | 10-100万 | 中强 | 冲销量 | 中（纯佣金或低坑位费） |
| **尾部达人** | 1-10万 | 中 | 测品 | 低（纯佣金） |
| **素人达人** | 1000-1万 | 弱 | 铺量 | 无（纯佣金） |

### 2.2 达人选择标准

#### 基础筛选条件

| 条件 | 说明 | 最低要求 |
|------|------|----------|
| **粉丝数** | 账号粉丝总量 | ≥10,000 |
| **视频数** | 近30天发布视频数 | ≥10条 |
| **互动率** | 点赞+评论/粉丝数 | ≥3% |
| **带货经验** | 有无带货视频 | 优先有经验 |
| **内容风格** | 与商品匹配度 | 高相关 |

#### 进阶筛选条件

| 条件 | 说明 | 评估方法 |
|------|------|----------|
| **历史带货数据** | 过往带货转化率 | 查看带货视频数据 |
| **粉丝画像** | 粉丝年龄、性别、地域 | TikTok Analytics |
| **内容质量** | 视频制作水平 | 观看近10条视频 |
| **粉丝活跃度** | 粉丝互动频率 | 评论活跃度 |
| **账号健康度** | 有无违规记录 | 检查账号状态 |

### 2.3 不同阶段达人选择策略

| 阶段 | 达人类型 | 策略 |
|------|----------|------|
| **冷启动期** | 尾部+素人 | 批量寄样，测试商品 |
| **成长期** | 尾部+腰部 | 筛选高效达人，深度合作 |
| **稳定期** | 腰部+头部 | 打造爆款，提升GMV |
| **爆发期** | 头部+明星 | 品牌曝光，大促冲量 |

## 四、合作模式详解

### 4.1 纯佣金模式

**定义**：商家不支付坑位费，仅按实际销售额支付佣金。

**适用场景**：
- 新手商家预算有限
- 测试新商品
- 合作尾部/素人达人

**佣金比例**：

| 商品类型 | 建议佣金比例 | 说明 |
|----------|-------------|------|
| 时尚配饰 | 18-22% | 标准佣金 |
| 美妆个护 | 18-22% | 标准佣金 |
| 3C配件 | 25-30% | 毛利高可让利 |
| 家居用品 | 15-20% | 毛利低控制成本 |

**优势**：
- 零风险，按效果付费
- 适合大量合作达人
- 快速测试商品

**劣势**：
- 达人合作意愿可能较低
- 头部达人可能不感兴趣
- 需要大量筛选才能找到高效的

### 4.2 坑位费+佣金模式

**定义**：商家支付固定坑位费 + 按销售额支付佣金。

**适用场景**：
- 合作腰部/头部达人
- 大促期间冲量
- 希望达人专门制作高质量视频

**坑位费参考**：

| 达人类型 | 坑位费范围 | 佣金比例 |
|----------|-----------|----------|
| 腰部达人 | $50-200 | 15-20% |
| 头部达人 | $200-1000 | 10-15% |

**优势**：
- 达人更愿意合作
- 视频质量有保障
- 可要求专属定制

**劣势**：
- 需要预付成本
- 存在风险（可能效果不好）
- 资金压力大

### 4.3 长期合作模式

**定义**：与达人建立长期合作关系，签订合作协议。

**适用场景**：
- 找到高效达人后
- 希望稳定带货来源
- 打造专属带货渠道

**合作内容**：
- 固定佣金比例（通常比单次更低）
- 独家合作（特定商品或类目）
- 月度最低视频数量
- 优先使用新品

**优势**：
- 佣金成本更低
- 供货有保障
- 达人更用心

**劣势**：
- 灵活性降低
- 可能错过其他达人
- 管理成本增加

## 五、佣金谈判技巧

### 5.1 谈判前准备

**必备信息**：
- 商品成本和利润空间
- 目标佣金比例
- 竞争对手佣金水平
- 达人历史数据

**谈判目标**：
- 底线佣金比例
- 理想佣金比例
- 可接受范围

### 5.2 谈判话术模板

#### 首次联系模板

```
你好！我是[TikTok Shop店铺名]的运营负责人。

我们在TikTok Shop销售[商品类目]，看到您最近在带[相关品类]，效果非常好！

我们有一款[商品名称]非常适合您的粉丝群，想和您合作：
- 商品：[商品链接]
- 佣金比例：20%
- 样品：我们免费提供

如果您感兴趣，我们可以详细聊聊合作细节～

期待您的回复！
```

#### 佣金谈判模板

**场景1：达人要求高佣金**

```
感谢您的合作意向！

关于佣金比例，我们希望控制在18%，因为[商品成本]的原因。

不过我们可以提供：
- 免费样品（价值$XX）
- 优先供货保障
- 月度合作机会
- 如果效果好，可以提高到20%

您看这样可以吗？
```

**场景2：达人嫌佣金低**

```
理解您的顾虑！

我们可以适当提高佣金：
- 第一个月20%（试用期）
- 效果好的话，第二个月开始22%
- 额外提供新品优先测试权

另外，如果您愿意独家合作，我们可以再提高2%。

期待和您长期合作！
```

### 5.3 谈判注意事项

| 注意事项 | 说明 |
|----------|------|
| **不要一上来就答应** | 先了解达人期望，再慢慢谈 |
| **强调双赢** | 强调合作对双方都有好处 |
| **留有余地** | 先报低价，留出谈判空间 |
| **抓住痛点** | 了解达人最看重什么（钱/样品/曝光） |
| **及时止损** | 如果达人不配合，换下一个 |

## 六、寄样流程详解

### 6.1 样品准备

#### 样品选择

| 原则 | 说明 |
|------|------|
| **高质量** | 选择质量最好的样品 |
| **有库存** | 确保有现货可发 |
| **包装完整** | 含包装、说明书、配件 |
| **价值合适** | 样品价值$10-30合适 |

#### 样品数量

| 达人类型 | 样品数量 | 说明 |
|----------|---------|------|
| 尾部达人 | 1-2件 | 测试为主 |
| 腰部达人 | 2-3件 | 可多寄几款 |
| 头部达人 | 3-5件 | 确保有选择 |

### 6.2 寄样流程

#### 步骤1：确认合作

- 达人接受合作邀请
- 确认寄样地址和联系方式
- 签订合作协议（可选）

#### 步骤2：准备样品

- 挑选高质量样品
- 拍摄样品照片留存
- 包装样品（含防护）

#### 步骤3：发货

- 选择可靠物流
- 填写运单号
- 记录发货时间

#### 步骤4：跟踪物流

- 监控物流进度
- 提醒达人查收
- 处理物流异常

#### 步骤5：确认收货

- 达人确认收货
- 感谢达人合作
- 确认视频发布时间

### 6.3 寄样模板消息

#### 寄样通知模板

```
您好！您的合作样品已发货啦～

📦 样品：[商品名称]
🚚 快递：[快递公司]
📦 单号：[运单号]
⏰ 预计：[到达时间]

收到后麻烦告知一下哦～

期待您的精彩视频！有任何问题随时联系我 😊
```

#### 催促进度模板

```
您好！想问一下样品的视频进展如何了呢？

📅 如果近期方便的话，希望能尽快安排拍摄～

如果还没收到样品，也请告诉我，我来帮您查询物流 🙏

期待您的爆款视频！
```

## 十、常见问题与解决方案

### Q1：达人收到样品不回复怎么办？

**解决方案**：
1. 等待3-5个工作日
2. 发送温馨提醒
3. 如果仍无回复，换下一个达人

### Q2：达人视频效果不好怎么办？

**解决方案**：
1. 分析视频数据，找出原因
2. 提供视频优化建议
3. 调整佣金激励
4. 更换合作达人

### Q3：如何避免达人跑单？

**解决方案**：
1. 选择有带货经验的达人
2. 签订简单合作协议
3. 及时跟进和沟通
4. 保持良好合作关系

### Q4：佣金比例怎么定？

**参考标准**：
- 商品毛利50%+：佣金20-25%
- 商品毛利30-50%：佣金15-20%
- 商品毛利<30%：佣金10-15%

### Q5：样品寄出后没收到怎么办？

**解决方案**：
1. 查询物流信息
2. 联系快递公司
3. 如丢失，重新寄送
4. 记录问题，避免再次发生
