# TikTok Shop 冷启动策略知识库

> 更新时间：2026年6月20日

## 一、冷启动底层逻辑

### 核心目标
**测出单模型**：全力撬动平台曝光 → 验证曝光和下单的转化关系 → 搭建可复制的稳定出单模型

### 关键指标
- **转化公式**：播放量 × 转化率 = 订单数
- **目标**：找到稳定的转化比例（如2万播放出1单）

---

## 二、60天破零计划

### 第1-15天：打底测款

**核心目标**：不求出单，只做数据铺垫和素材积累

| 渠道 | 任务 | 每日预算 |
|------|------|----------|
| 商品卡 | 持续优化产品主图、标题及详情页 | 25美金 |
| 自营短视频 | 坚持日更3-5条，纯做自然流量起量 | 0 |
| 达人合作 | 批量寄出约100份样品 | 0 |
| 直播 | 直接暂缓启动 | - |

**目标**：15天沉淀80条左右原创素材

### 第16-30天：投流实测

**核心目标**：对比各渠道转化效率，找到最优打法

| 渠道 | 任务 | 每日预算 |
|------|------|----------|
| 商品卡 | 保持页面日常优化，维持每日投流 | 25美金 |
| 自营短视频 | 素材库累积至80-100条，开启小额投流 | 25美金 |
| 达人合作 | 回收5-10条带专属链接的达人视频 | 0 |
| 直播 | 有团队可小成本试水 | - |

**目标**：筛选高效素材，准备下一阶段放大

### 第31-45天：稳定出单

**核心目标**：店铺正式步入稳定出单

| 渠道 | 任务 | 每日预算 |
|------|------|----------|
| 商品卡 | 维持原有投流节奏和页面优化 | 25美金 |
| 自营短视频 | 素材库扩充至200-300条，筛选10条潜力爆款 | 25美金 |
| 达人合作 | 累计回收20-40条达人带货视频 | 0 |
| 直播 | 转化达标就持续运营 | - |

**目标**：日更10条，夯实曝光转化链路

### 第46-60天：模型固化

**核心目标**：打造可复制、可放大的标准化出单模型

| 渠道 | 任务 | 每日预算 |
|------|------|----------|
| 商品卡 | 保持固定预算和更新节奏 | 25美金 |
| 自营短视频 | 日更10条，沉淀200+素材 | 25美金 |
| 达人合作 | 达人带货视频累计沉淀80条以上 | 0 |
| 直播 | 聚焦核心渠道发力 | - |

**目标**：完整跑通曝光→点击→加购→下单全链路

## 三、四大出单渠道优先级

| 优先级 | 渠道 | 说明 |
|--------|------|------|
| 1 | 商品卡 | 日常优化+小额投流（最重要） |
| 2 | 自营短视频 | 日更3-10条（核心） |
| 3 | 达人合作 | 批量寄样+筛选优质（加速） |
| 4 | 直播 | 短时高频30-60分钟/次（进阶） |

---

## 四、新手必守3条黄金原则

### 1. 严控预算，拒绝盲目烧钱
- 单渠道每日预算控制在25美金
- 重在测试转化效率，而非堆砌投放金额

### 2. 素材以量取胜
- 从日更3-5条逐步递增至10条
- 靠海量作品筛选优质内容
- 别寄希望于一两支视频爆火

### 3. 达人广撒网、精深耕
- 前期批量寄样100份，不催促即时出稿
- 筛选出5-10个高转化达人深度合作
- 建立长期合作关系，降低合作成本

## 六、失败应对策略

### 如果60天没有任何正向数据
- ❌ 别死磕浪费时间
- ✅ 果断换品重启
- ✅ 做跨境，选品和时机远比盲目死守更重要
