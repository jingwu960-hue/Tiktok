## 二、定价公式

### 基础公式
```
售价 = 采购成本 ÷ (1 - 平台费率 - 达人佣金 - 广告费用 - 退货损耗 - 汇率损耗 - 目标利润率)
```

### 简化公式
```
售价 = 采购价 × 4 ~ 5
```

### 达人带货定价公式
```
售价 = 采购成本 ÷ (1 - 平台费率 - 达人佣金 - 目标利润率)
```

**示例**（采购价20元，达人佣金20%，平台费率21.5%，目标利润率25%）：
```
售价 = 20 ÷ (1 - 0.215 - 0.20 - 0.25)
     = 20 ÷ 0.335
     = 59.7元 ≈ 60元
```

## 五、利润计算工具

### 在线计算器
```
售价（元）= [采购价 + 其他成本] ÷ (1 - 平台费率 - 目标利润率)
```

### 示例计算
**假设**：
- 采购价：20元
- 平台费率：21.5%
- 目标利润率：25%

**计算**：
```
售价 = 20 ÷ (1 - 0.215 - 0.25) = 20 ÷ 0.535 = 37.4元
```
