## 一、广告投放概述

### 1.1 为什么需要广告投放

广告投放是放大自然流量的重要手段，特别是在TikTok Shop运营中：

| 优势 | 说明 |
|------|------|
| **快速起量** | 付费流量快速提升曝光 |
| **精准触达** | 定向投放目标用户 |
| **效果可控** | 按效果付费，ROI可追踪 |
| **放大爆款** | 让爆款更爆 |
| **数据验证** | 测试商品和素材 |

### 1.2 广告投放核心指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| **CPC** | 单次点击成本 | $0.02-0.08 |
| **CPM** | 千次曝光成本 | $2-5 |
| **CTR** | 点击率 | ≥2% |
| **ROAS** | 广告回报率 | ≥1:3 |
| **转化率** | 点击购买率 | ≥1% |

## 二、广告类型详解

### 2.1 商品推广（Product Promotion）

**定义**：推广单个商品，吸引用户购买。

**适用场景**：
- 推广爆款商品
- 新品推广
- 清仓促销

**优势**：
- 转化路径短
- 数据清晰
- 优化简单

**设置要点**：

| 设置项 | 建议 |
|--------|------|
| **推广商品** | 选择有库存、有流量的商品 |
| **推广目标** | 选择"转化量" |
| **出价方式** | CPC或oCPM |
| **定向** | 按年龄、性别、地域定向 |

### 2.2 视频推广（Video Promotion）

**定义**：推广带货视频，增加视频曝光。

**适用场景**：
- 推广表现好的视频
- 测试新素材
- 增加账号曝光

**优势**：
- 提升视频播放量
- 增加粉丝
- 软性带货

**设置要点**：

| 设置项 | 建议 |
|--------|------|
| **推广视频** | 选择完播率高的视频 |
| **推广目标** | 选择"视频播放量"或"转化量" |
| **出价方式** | CPM或oCPM |
| **优化方向** | 播放量/点击/转化 |

### 2.3 Brand hashtags（品牌话题）

**定义**：推广品牌话题标签，提升品牌曝光。

**适用场景**：
- 品牌宣传
- 话题营销
- 活动推广

**优势**：
- 品牌曝光
- 用户参与
- 长期价值

## 三、GMV Max工具使用教程

### 3.1 GMV Max概述

**定义**：TikTok Shop提供的智能投放工具，帮助商家提升GMV。

**核心功能**：
- 系统自动优化投放
- 全店GMV提升
- 智能预算分配

### 3.2 开通条件

| 条件 | 要求 | 说明 |
|------|------|------|
| **店铺评分** | ≥4.0 | 综合评分 |
| **订单量** | 近30天≥100单 | 基础门槛 |
| **类目** | 需申请开通 | 部分类目 |

### 3.3 操作教程

#### 步骤1：进入GMV Max

**路径**：TikTok Shop卖家中心 → 营销中心 → GMV Max

#### 步骤2：创建投放计划

```
1. 点击"创建计划"
2. 选择投放目标（GMV提升/订单量）
3. 设置日预算
4. 选择投放时长
5. 确认投放
```

#### 步骤3：设置预算

| 预算类型 | 说明 | 建议 |
|----------|------|------|
| **日预算** | 每日最高花费 | $20-50起 |
| **总预算** | 整个计划总花费 | 日预算×天数 |
| **出价** | 系统自动出价 | 建议开启智能出价 |

#### 步骤4：优化设置

| 设置项 | 选项 | 说明 |
|--------|------|------|
| **优化目标** | GMV/订单量 | 选择核心目标 |
| **定向** | 系统智能定向 | 开启自动定向 |
| **排期** | 持续投放/定时 | 按需选择 |

### 3.4 GMV Max优势

| 优势 | 说明 |
|------|------|
| **智能优化** | 系统自动优化投放效果 |
| **全店提升** | 不仅推广单品，提升全店GMV |
| **操作简单** | 只需设置预算，其他交给系统 |
| **数据透明** | 实时查看投放效果 |

## 四、Smart Promotion项目

### 4.1 Smart Promotion概述

**定义**：TikTok Shop的智能营销项目，参与可获得平台资源扶持。

**参与收益**：
- 平台流量补贴
- 活动资源位
- 专属标签
- 优先推荐

### 4.2 参与条件

| 条件 | 要求 | 说明 |
|------|------|------|
| **店铺体验分** | ≥3.5分 | 综合评分 |
| **订单量** | 近90天≥30单 | 基础门槛 |
| **店铺健康分** | >150分 | 店铺健康度 |
| **类目** | 部分类目受限 | 需确认是否可参与 |

### 4.3 参与方式

#### 步骤1：检查资格

```
卖家中心 → 营销中心 → Smart Promotion → 查看资格
```

#### 步骤2：报名参加

```
点击"立即报名" → 确认参与 → 等待审核
```

#### 步骤3：完成日常任务

| 任务类型 | 说明 | 奖励 |
|----------|------|------|
| **日常任务** | 每日完成基础操作 | 平台流量补贴 |
| **GMV任务** | 完成GMV目标 | 额外资源 |
| **活动任务** | 参与平台活动 | 活动曝光 |

### 4.4 Smart Promotion收益

| 时期 | 收益 |
|------|------|
| **日常期** | 平台发放≥3.5%折扣资源 |
| **大促期** | 平台发放≥4.5%折扣资源 |
| **专属标签** | 商品获得"Smart"标签 |
| **活动直通** | 直通所有平台级活动 |

## 五、出价策略详解

### 5.1 CPC出价策略

**定义**：按点击付费，每次点击收取固定费用。

**适用场景**：
- 商品推广
- 追求点击量
- 测试素材

**出价参考**：

| 市场 | CPC范围 | 建议出价 |
|------|---------|----------|
| 泰国 | $0.03-0.08 | $0.05 |
| 越南 | $0.02-0.06 | $0.04 |
| 马来西亚 | $0.03-0.07 | $0.05 |
| 菲律宾 | $0.02-0.05 | $0.03 |

### 5.2 CPM出价策略

**定义**：按千次曝光付费。

**适用场景**：
- 品牌曝光
- 视频推广
- 追求曝光量

**出价参考**：

| 市场 | CPM范围 | 建议出价 |
|------|---------|----------|
| 泰国 | $3-6 | $4 |
| 越南 | $2-5 | $3 |
| 马来西亚 | $3-6 | $4 |
| 菲律宾 | $2-5 | $3 |

### 5.3 oCPM出价策略

**定义**：优化千次曝光付费，系统自动优化转化。

**适用场景**：
- 追求转化
- 自动优化
- 效果导向

**出价设置**：

| 优化目标 | 建议出价 | 说明 |
|----------|---------|------|
| **转化量** | 目标CVR×目标CPC | 根据商品价格设置 |
| **点击量** | 目标CTR×目标CPM/1000 | 根据市场调整 |
| **曝光量** | 市场CPM×1.1-1.2 | 略高于市场均价 |

### 5.4 预算分配策略

| 策略 | 适用场景 | 分配比例 |
|------|----------|----------|
| **测试期** | 新品测试 | 测试70%/放量30% |
| **放量期** | 效果验证 | 测试30%/放量70% |
| **稳定期** | 持续投放 | 固定预算/AB测试 |

## 六、广告素材制作

### 6.1 图片素材要求

| 要求 | 说明 |
|------|------|
| **尺寸** | 1:1或9:16 |
| **像素** | 1080×1080或1080×1920 |
| **格式** | JPG/PNG |
| **大小** | ≤5MB |

### 6.2 视频素材要求

| 要求 | 说明 |
|------|------|
| **时长** | 5-60秒 |
| **尺寸** | 9:16（竖版最佳） |
| **格式** | MP4 |
| **大小** | ≤500MB |

### 6.3 素材创意要点

| 要素 | 说明 | 示例 |
|------|------|------|
| **强开场** | 前3秒抓住注意力 | "这个东西太神奇了！" |
| **痛点** | 解决用户问题 | "还在为XX烦恼？" |
| **展示** | 展示产品效果 | 使用前后对比 |
| **促销** | 提供优惠信息 | "现在只要XX！" |
| **CTA** | 明确的行动召唤 | "点击购买" |

## 七、数据分析与优化

### 7.1 核心数据指标

| 指标 | 计算方法 | 优化方向 |
|------|----------|----------|
| **曝光** | 广告展示次数 | 提升出价/优化定向 |
| **点击** | 用户点击次数 | 优化素材/标题 |
| **CTR** | 点击/曝光 | 提升创意质量 |
| **转化** | 用户购买数 | 优化落地页/价格 |
| **CVR** | 转化/点击 | 提升商品吸引力 |
| **CPC** | 花费/点击 | 优化出价 |
| **CPM** | 花费/曝光×1000 | 优化出价 |
| **ROAS** | GMV/广告花费 | 提升整体效率 |

### 7.2 数据分析流程

```
1. 查看数据 → 2. 识别问题 → 3. 制定优化方案 → 4. 执行测试 → 5. 验证效果
```

#### 数据诊断表

| 问题 | 指标表现 | 优化方向 |
|------|----------|----------|
| **曝光少** | 曝光<预期50% | 提高出价/放宽定向 |
| **点击低** | CTR<1% | 优化素材/标题 |
| **转化低** | CVR<0.5% | 优化价格/详情页 |
| **ROI低** | ROAS<1:2 | 控制成本/提升转化 |

### 7.3 优化技巧

| 阶段 | 优化重点 | 具体操作 |
|------|----------|----------|
| **测试期** | 素材测试 | 多套素材AB测试 |
| **放量期** | 成本控制 | 优化出价和定向 |
| **稳定期** | 效率提升 | 持续优化素材 |

## 九、常见问题

### Q1：广告费花不出去怎么办？

**解决方案**：
1. 提高出价
2. 放宽定向
3. 更换投放时间
4. 优化素材质量

### Q2：广告ROI很低怎么办？

**解决方案**：
1. 检查商品价格是否合理
2. 优化落地页
3. 调整定向人群
4. 更换素材
5. 控制出价

### Q3：如何判断广告效果好不好？

**判断标准**：
- ROAS ≥ 1:3 ✅
- ROAS 1:2-3 ⚠️ 需优化
- ROAS < 1:2 ❌ 需调整

### Q4：预算怎么分配？

**参考方案**：
- 测试预算：总预算30%
- 主力投放：总预算50%
- 机动预算：总预算20%

### Q5：什么时候开始投广告？

**建议时机**：
- 商品有自然流量后
- 素材测试完成
- 预算充足
- 运营稳定后

## 十、PEAKS方法论 - Activities & Advertising（营销放大）

> 来源：TikTok Shop美区跨境POP PEAKS出海经营方法论白皮书

### 10.1 营销放大战略定位

**营销放大是PEAKS方法论的增长加速器**

通过活动和广告投放，放大优质内容和商品的曝光，实现销售增长。

| 战略价值 | 说明 |
|---------|------|
| **流量放大** | 付费流量快速提升曝光 |
| **精准触达** | 定向投放目标用户 |
| **效果可控** | 按效果付费，ROI可追踪 |
| **爆款加速** | 让爆款更爆 |

### 10.2 营销活动类型

#### 平台活动

| 活动类型 | 频率 | 参与条件 | 收益 |
|---------|------|---------|------|
| **大促活动** | 月度/季度 | 店铺评分≥4.0 | 平台流量补贴 |
| **品类活动** | 每周 | 类目相关 | 品类曝光 |
| **品牌活动** | 不定期 | 品牌商家 | 品牌曝光 |
| **新人活动** | 持续 | 新店铺 | 新店扶持 |

#### 自主活动

| 活动类型 | 适用场景 | 预算占比 |
|---------|---------|---------|
| **限时折扣** | 清仓、冲量 | 10-15% |
| **满减优惠** | 提升客单价 | 5-10% |
| **买赠活动** | 新品推广 | 5-10% |
| **粉丝专享** | 粉丝运营 | 3-5% |

### 10.3 广告投放策略

#### 投放阶段规划

| 阶段 | 目标 | 预算分配 | 出价策略 |
|------|------|---------|---------|
| **测试期** | 验证素材和商品 | 总预算30% | 低出价测试 |
| **放量期** | 放大有效投放 | 总预算50% | 中高出价 |
| **稳定期** | 持续优化ROI | 总预算20% | 智能出价 |

#### 投放渠道选择

| 渠道 | 适用场景 | 预算占比 | 预期ROI |
|------|---------|---------|---------|
| **GMV Max** | 全店GMV提升 | 40% | 1:3-5 |
| **商品推广** | 单品爆款打造 | 30% | 1:2-4 |
| **视频推广** | 内容曝光 | 20% | 1:1-3 |
| **达人合作** | 内容放量 | 10% | 1:3-6 |

### 10.4 营销数据化运营

#### 核心数据指标

| 指标 | 计算方法 | 目标值 | 优化方向 |
|------|----------|--------|---------|
| **ROAS** | GMV/广告花费 | ≥1:3 | 提升转化效率 |
| **CPO** | 广告花费/订单数 | ≤$5 | 降低获客成本 |
| **CPM** | 花费/曝光×1000 | $2-5 | 优化出价 |
| **CTR** | 点击/曝光 | ≥2% | 优化素材 |

#### 数据监测频率

| 监测维度 | 频率 | 关键动作 |
|---------|------|---------|
| **实时监测** | 每小时 | 发现异常，及时调整 |
| **日常监测** | 每日 | 分析数据，优化投放 |
| **周度复盘** | 每周 | 总结经验，调整策略 |
| **月度规划** | 每月 | 制定下月计划 |

### 10.5 营销放大SOP

#### 大促活动SOP

| 阶段 | 时间 | 任务 | 目标 |
|------|------|------|------|
| **预热期** | 活动前7天 | 选品、备货、素材准备 | 充分准备 |
| **爆发期** | 活动期间 | 加大投放、及时发货 | 冲刺销量 |
| **返场期** | 活动后3天 | 维持热度、处理售后 | 延长效果 |
| **复盘期** | 活动后7天 | 数据分析、总结经验 | 持续优化 |
