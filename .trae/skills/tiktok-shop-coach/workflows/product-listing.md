# 商品上架优化工作流程

## 触发条件

用户询问"上架"、"标题"、"SEO"、"卖点"、"视频创意"等商品优化相关问题时触发。

## 执行步骤

### 1. 识别优化需求
判断用户需要哪方面的帮助：
- 新商品上架（完整流程）
- 标题优化
- 主图优化
- 详情页优化
- 短视频创意

### 2. 新商品上架流程

#### 2.1 主图准备
- **尺寸**：800×800px 以上，正方形，1200×1200 最佳
- **背景**：纯白（RGB 255,255,255），无水印、无文字
- **占比**：产品占画面80%以上
- **数量**：1张主图 + 4张辅图（场景图、细节图、尺寸图、使用图）

#### 2.2 规格参数
- 基础信息：材质、尺寸、重量、颜色
- 功能特点：核心功能、适用场景、优点
- 包装清单：包含配件、数量
- 注意：中文填写即可，平台自动翻译

#### 2.3 标题公式
```
核心关键词 + 产品属性 + 功能特点 + 适用场景
```

**示例**：
- 太阳镜：复古椭圆金属太阳镜 防紫外线 时尚百搭
- 戒指：六字真言旋转男戒 时尚个性 送礼佳品
- 手表：米兰网带女表 简约气质 送礼首选

#### 2.4 定价设置
参考 `config/parameters.yaml` 中的 `pricing` 参数。

#### 2.5 详情页
- 产品卖点（3-5个）
- 使用场景展示
- 尺寸/规格说明
- 包装清单
- 售后说明

### 3. 标题优化

**优化原则**：
- 包含核心关键词（越南语）
- 突出产品差异化卖点
- 长度适中（不超过80字符）
- 避免堆砌关键词

**优化步骤**：
1. 分析当前标题问题
2. 研究竞品标题结构
3. 使用标题公式重写
4. 检查关键词覆盖度

### 4. 主图优化

**优化原则**：
- 白底、清晰、高质感
- 产品占比大
- 避免中文水印
- 辅图展示使用场景

### 5. 短视频创意

**爆款结构（15秒）**：
```
0-3秒：痛点钩子
3-8秒：解法演示
8-15秒：行动指令
```

**三种模板**：
1. 前后对比法
2. 场景痛点法
3. 反常识惊喜法

**发布节奏**：
- 最佳时长：15-25秒
- 完播率目标：>40%
- 更新频率：日更3-10条

## 错误处理

### 场景1：用户无产品图片
**处理方式**：
1. 建议从1688/拼多多获取
2. 使用美图秀秀处理白底图
3. 提供供应商联系模板索取素材

### 场景2：标题翻译困难
**处理方式**：
1. 先用中文写出核心卖点
2. 使用翻译工具转为越南语
3. 检查语法和表达是否自然

## 输出格式

根据具体需求输出：
- 新商品上架清单
- 标题优化方案
- 主图优化建议
- 短视频脚本

## 相关参数

参考 `config/parameters.yaml` 中的：
- `pricing` - 定价公式
- `short_video` - 短视频参数
- `supplier` - 供应商沟通参数
