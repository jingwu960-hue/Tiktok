# 选品分析工作流程

## 触发条件

用户询问"选品"、"卖什么"、"利润分析"、"竞争分析"、"定价"、"成本"等选品相关问题时触发。

## 执行步骤

### 1. 收集产品信息
向用户收集以下信息：
- 产品名称
- 采购价（CNY）
- 重量（g）
- 尺寸（cm）
- 目标售价（可选）

### 2. 验证选品铁律
根据 `config/parameters.yaml` 中的 `selection_rules` 验证：

| 规则 | 阈值 | 验证结果 |
|------|------|----------|
| 重量 | <200g | ✅/❌ |
| 体积 | <20×15×10cm | ✅/❌ |
| 采购价 | ≤20元 | ✅/❌ |
| 客单价 | $5-$15 | ✅/❌ |

**任一规则不通过**：给出警告，说明风险。

### 3. 计算利润
使用定价公式计算：

```
总成本 = 采购价 + 跨境运费 + 平台佣金
利润 = 售价 - 总成本
利润率 = 利润 / 售价 × 100%
```

**参数来源**：
- 跨境运费：`cost_parameters.shipping_rate`（2.3-3元/kg）
- 平台佣金：`cost_parameters.commission_rate`（6%）
- 汇率：`cost_parameters.exchange_rate`（1 CNY ≈ 3,872 VND）

### 4. 市场分析
评估以下维度：
- **市场需求**：高/中/低（根据品类和季节判断）
- **竞争程度**：红海/蓝海/中等（根据店铺数判断）
- **复购潜力**：高/中/低（根据产品特性判断）

### 5. 综合评分
根据利润率、竞争度、复购潜力综合评分：

| 利润率 | 竞争度 | 复购潜力 | 评分 |
|--------|--------|----------|------|
| ≥30% | 蓝海 | 高 | ★★★★★ |
| ≥30% | 中等 | 中 | ★★★★☆ |
| 25-30% | 中等 | 中 | ★★★☆☆ |
| <25% | 红海 | 低 | ★★☆☆☆ |

### 6. 输出报告
使用 `config/templates/product-analysis.md` 模板生成报告。

### 7. 给出建议
- **推荐**：说明理由和下一步行动
- **不推荐**：说明风险和改进方向

## 错误处理

### 场景1：用户未提供采购价
**处理方式**：
1. 询问用户采购价
2. 如用户不确定，提供1688/拼多多参考价格
3. 使用估算值时明确标注

### 场景2：利润率计算结果<25%
**处理方式**：
1. 警告用户利润率偏低
2. 建议调整售价或寻找更低成本供应商
3. 提供优化建议

### 场景3：产品重量>200g
**处理方式**：
1. 警告物流成本过高
2. 计算实际运费影响
3. 建议寻找更轻的替代品

## 输出格式

使用 `config/templates/product-analysis.md` 模板输出。

## 示例

**用户输入**："帮我分析一下太阳镜，采购价15元，重量50g"

**执行流程**：
1. 收集信息：太阳镜，15元，50g
2. 验证铁律：重量✅，采购价✅
3. 计算利润：建议售价45元，利润率33%
4. 市场分析：需求高，竞争中等，复购中
5. 综合评分：★★★★☆
6. 输出报告

**输出**：
```markdown
# 🔍 选品分析报告：时尚太阳镜

## 产品信息
- **采购价**：15元
- **建议售价**：45元（约174,240 VND）
- **利润率**：33%

## 市场分析
- **市场需求**：高（夏季刚需）
- **竞争程度**：中等（店铺数约80家）
- **复购潜力**：中（季节性购买）

## 利润测算
| 成本项 | 金额（CNY） | 金额（VND） |
|--------|-------------|-------------|
| 采购价 | 15 | 58,080 |
| 跨境运费 | 3 | 11,616 |
| 平台佣金 | 2.7 | 10,454 |
| 总成本 | 20.7 | 80,150 |
| 利润 | 24.3 | 94,090 |

## 综合评分
★★★★☆

## 建议
**推荐**
理由：利润率达标，市场需求稳定，竞争可控。建议作为夏季主推款。
```

## 相关参数

参考 `config/parameters.yaml` 中的：
- `selection_rules` - 选品铁律
- `cost_parameters` - 成本参数
- `pricing` - 定价公式
