# 抖音口播脚本模板

## 使用说明

此模板用于将当日运营日志转化为抖音口播脚本。脚本定位为**程序员 + 跨境电商新手创业者的真实记录**，风格真诚务实。

**输入**：当日日志文件（如 `TikTokShop/2026/6月/6月22日.md`）
**输出**：口播脚本文件（如 `TikTokShop/2026/6月/脚本/6月22日-脚本.md`）

---

## 一、脚本元信息

```markdown
# {YYYY}年{M}月{D}日 抖音口播脚本

> 时长：约 {N} 分钟 | 字数：约 {N} 字 | 风格：真诚务实

---
```

---

## 二、脚本结构

### 模块①：钩子（0-15秒，约40-60字）

**作用**：一句话抓住注意力，让观众愿意继续看下去。

**风格要求**：
- 像朋友聊天，平实但有力
- 用真实数据或事件说话
- 不超过 30 字

**禁止**：
- 标题党（"震惊！""竟然！""你绝对想不到"）
- 夸张数字（"暴富""日入过万"）
- 虚假承诺（"保证赚钱"）
- 焦虑贩卖（"再不行动就晚了"）

**示例**：
- "今天是我做 TikTok 越南店的第 4 天，我把一个商品下架了。"
- "5000 块预算做跨境电商，今天遇到一个坎。"
- "程序员做跨境电商的第 4 天，踩了一个坑。"

**模板**：
```markdown
### 钩子

{一句话概括今天最核心的事件或数据变化，真诚务实}
```

---

### 模块②：今日故事（15-90秒，约100-250字）

**作用**：讲述今天发生了什么，让观众有代入感。

**结构**：背景 → 转折 → 结果 → 感悟

**素材来源**：
- 优先使用日志中 `🎬 今日故事` 模块
- 如不存在，从 `每日复盘` + `关键决策` 中重构

**模板**：
```markdown
### 今日故事

今天是我做 TikTok 越南店的第 {N} 天。

{背景}
{转折}
{结果}

说实话，{感悟}。
```

**示例**：
```markdown
### 今日故事

今天是我做 TikTok 越南店的第 4 天。

上午照常检查店铺，发现一个 Mickey 手表商品。我突然意识到，这种带迪士尼 IP 的商品可能涉及侵权。

查了一下平台规则，果然——品牌侵权是红线，轻则下架，重则封店。

我立刻把它下架了。虽然少了一个商品，但心里踏实了很多。

说实话，新手做跨境电商，不懂规则比不懂运营更可怕。
```

---

### 模块③：干货拆解（90-210秒，约250-450字）

**作用**：从今天的经历中提炼一个可复用的方法论，提供价值。

**素材来源**：
- 优先从 `问题诊断` / `思考过程` / `关键决策` 中提取
- 如无，可从 `学习要点` 中提取

**结构**：
1. 问题是什么（一句话）
2. 为什么重要（对新手的影响）
3. 怎么做（2-4 个具体步骤）
4. 避坑提醒（1-2 个注意事项）

**程序员视角融入**（仅在相关时使用）：
- 数据抓取/分析 → "我写了个爬虫..."
- 自动化工具 → "用 Python 写了个脚本..."
- 效率提升 → "程序员思维告诉我..."
- 系统化方法 → "我把它抽象成一个模型..."

**模板**：
```markdown
### 干货拆解

今天我想分享一个经验：{主题}

{问题是什么}

{为什么重要}

{怎么做 - 步骤1}
{怎么做 - 步骤2}
{怎么做 - 步骤3}

{避坑提醒}

{如果有程序员视角，在这里自然融入}
```

---

### 模块④：数据看板（210-240秒，约80-120字）

**作用**：口述今日运营数据，如实呈现，不做夸张解读。

**素材来源**：日志中 `运营数据` 表格

**模板**：
```markdown
### 数据看板

来看一下今天的数据：

| 指标 | 数值 |
|------|------|
| 商品数 | {N} |
| 订单数 | {N} |
| 视频数 | {N} |
| 粉丝数 | {N} |

{一句简洁的总结，不夸大，不焦虑}
```

**示例**：
```markdown
### 数据看板

来看一下今天的数据：商品 19 个，订单 0，视频 2 个，粉丝 0。

我知道数据不好看，但这是真实的。冷启动阶段就是这样，一步一个脚印。
```

---

### 模块⑤：结尾钩子（240-270秒，约60-80字）

**作用**：预告明天内容，引导互动，金句收尾。

**素材来源**：
- 预告：日志中 `明日计划` 核心任务
- 金句：日志中 `💡 视频素材标记` → `金句`

**模板**：
```markdown
### 结尾

{明日预告}

{互动引导}

{金句收尾}

---
```

**示例**：
```markdown
### 结尾

明天我准备开始养号，每天刷视频、点赞、评论，先把账号权重做起来。

如果你也在做 TikTok，有什么问题可以评论区聊聊。

创业不是冲刺，是马拉松。慢一点，稳一点。

---
```

## 四、完整脚本示例

```markdown
# 2026年6月22日 抖音口播脚本

> 时长：约 4 分钟 | 字数：约 800 字 | 风格：真诚务实

---

## 脚本内容

### 钩子

今天是我做 TikTok 越南店的第 4 天，我把一个商品下架了。

### 今日故事

上午照常检查店铺，发现一个 Mickey 手表商品。我突然意识到，这种带迪士尼 IP 的商品可能涉及侵权。

查了一下平台规则，果然——品牌侵权是红线，轻则下架，重则封店。

我立刻把它下架了。虽然少了一个商品，但心里踏实了很多。

说实话，新手做跨境电商，不懂规则比不懂运营更可怕。

### 干货拆解

今天我想分享一个经验：新手怎么判断商品有没有侵权风险。

侵权主要有三种：商标侵权、外观专利侵权、版权侵权。对于我们这种做时尚配件的，最容易踩的是商标侵权——比如用了迪士尼、漫威、耐克这些品牌的图案或文字。

我总结了三个自查方法：

第一，看商品标题和图片。有没有知名的品牌名、Logo、卡通形象？有的话，直接放弃。

第二，查 1688 供应商。如果供应商说"外贸原单""同款"，基本就是仿品，别碰。

第三，用平台规则对照。TikTok Shop 有禁售/限售商品清单，上架前先过一遍。

记住一句话：宁可少卖，不要违规。封店是一瞬间的事，但建店要花好几天。

### 数据看板

来看一下今天的数据：商品 19 个，订单 0，视频 2 个，粉丝 0。

我知道数据不好看，但这是真实的。冷启动阶段就是这样，一步一个脚印。

### 结尾

明天我准备开始养号，每天刷视频、点赞、评论，先把账号权重做起来。

如果你也在做 TikTok，有什么问题可以评论区聊聊。

创业不是冲刺，是马拉松。慢一点，稳一点。

---
```
