# 月度运营规划模板

## 使用说明

此模板用于生成月度/周度运营规划。根据店铺当前阶段和目标，制定可执行的运营计划。

## 模板结构

```markdown
# 📅 {月份}月度运营规划

## 核心目标
- GMV目标：{gmv_target}
- 订单目标：{orders_target}
- 商品数目标：{products_target}

## 周度拆解

### 第1周：{主题}
- 目标：{目标描述}
- 关键动作：{动作1、动作2}

### 第2周：{主题}
（同上）

### 第3周：{主题}
（同上）

### 第4周：{主题}
（同上）

## 资源分配
- 预算：{budget}元/天
- 时间：{hours}小时/天

## 风险预案
- 如果{风险点}未达成 → {应对方案}
```

## 变量说明

| 变量 | 来源 | 说明 |
|------|------|------|
| `{月份}` | 当前日期 | 规划月份 |
| `{gmv_target}` | 根据阶段设定 | 月度GMV目标 |
| `{orders_target}` | 根据阶段设定 | 月度订单目标 |
| `{products_target}` | 根据阶段设定 | 商品数量目标 |
| `{budget}` | 用户背景 | 每日预算 |
| `{hours}` | 用户背景 | 每日可用时间 |

## 阶段目标参考

| 阶段 | 天数 | GMV目标 | 订单目标 | 商品数 |
|------|------|---------|----------|--------|
| 冷启动 | 1-15天 | 0 | 0-5 | 20-30 |
| 成长期 | 16-30天 | 500,000 VND | 5-15 | 30-50 |
| 稳定期 | 31-60天 | 2,000,000 VND | 15-30 | 50-100 |

## 示例输出

```markdown
# 📅 7月度运营规划

## 核心目标
- GMV目标：5,000,000 VND（约1,290元）
- 订单目标：25单
- 商品数目标：50个

## 周度拆解

### 第1周：基础优化
- 目标：完成30个商品上架，优化主图
- 关键动作：
  1. 每天上架5-6个新品
  2. 优化现有商品主图
  3. 发布3条短视频/天

### 第2周：流量测试
- 目标：开启商品卡投流，测试CTR
- 关键动作：
  1. 每日投流$25
  2. 分析点击数据
  3. 优化低CTR商品

### 第3周：转化优化
- 目标：提升转化率到1.5%
- 关键动作：
  1. 优化商品详情页
  2. 设置优惠券
  3. 联系5个达人合作

### 第4周：数据复盘
- 目标：月度数据复盘，调整策略
- 关键动作：
  1. 分析整月数据
  2. 识别爆款潜力品
  3. 制定下月计划

## 资源分配
- 预算：50元/天
- 时间：5小时/天

## 风险预案
- 如果第2周CTR<2% → 暂停投流，重新优化主图
- 如果订单<10单 → 增加达人合作，扩大曝光
```
