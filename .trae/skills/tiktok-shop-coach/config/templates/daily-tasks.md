# 每日任务清单模板

## 使用说明

此模板用于生成每日运营指导报告。根据店铺当前状态和冷启动阶段，生成个性化的每日任务清单。

## 模板结构

```markdown
# 🎯 今日任务清单（第{N}天）

## 当前状态
- 商品数：{products_listed}
- 订单数：{orders}
- 阶段：{冷启动/成长期/稳定期}

## 今日最重要的3件事

### 任务1：{任务名称}
- **为什么做**：{原因说明}
- **具体步骤**：
  1. {步骤1}
  2. {步骤2}
  3. {步骤3}
- **预计耗时**：{N}分钟
- **完成标准**：{可验证的标准}
- **验证方法**：{如何确认完成}

### 任务2：{任务名称}
（同上格式）

### 任务3：{任务名称}
（同上格式）

## 失败调整
- 任务1失败 → {调整方案}
- 任务2失败 → {调整方案}

## 明日预告
- {预告内容}
```

## 变量说明

| 变量 | 来源 | 说明 |
|------|------|------|
| `{N}` | shop-metrics.json | 当前天数（day_number） |
| `{products_listed}` | shop-metrics.json | 已上架商品数 |
| `{orders}` | shop-metrics.json | 累计订单数 |
| `{阶段}` | 根据 day_number 判断 | 1-15天冷启动，16-30天成长期，31+天稳定期 |

## 生成逻辑

1. 读取 shop-metrics.json 获取当前状态
2. 根据 day_number 判断冷启动阶段
3. 从 workflows/daily-guide.md 加载对应阶段的任务库
4. 选择最重要的3个任务
5. 填充模板变量，生成最终报告

## 示例输出

```markdown
# 🎯 今日任务清单（第5天）

## 当前状态
- 商品数：20
- 订单数：0
- 阶段：冷启动期

## 今日最重要的3件事

### 任务1：优化商品主图
- **为什么做**：提升点击率，为后续投流做准备
- **具体步骤**：
  1. 选择5个核心商品
  2. 用美图秀秀处理白底图
  3. 上传替换原主图
- **预计耗时**：60分钟
- **完成标准**：5个商品主图全部替换
- **验证方法**：进入商品详情页确认图片已更新

### 任务2：发布3条短视频
- **为什么做**：积累素材，测试内容方向
- **具体步骤**：
  1. 选择3个爆款模板
  2. 制作15秒视频
  3. 添加越南语文案
- **预计耗时**：90分钟
- **完成标准**：3条视频成功发布
- **验证方法**：进入账号主页确认视频已发布

### 任务3：联系10个达人
- **为什么做**：建立达人合作渠道
- **具体步骤**：
  1. 在TikTok搜索相关达人
  2. 发送合作邀请
  3. 记录回复情况
- **预计耗时**：45分钟
- **完成标准**：发送10个邀请
- **验证方法**：检查私信发送记录

## 失败调整
- 任务1失败 → 优先完成任务2和3，主图优化延后
- 任务2失败 → 减少到1条，确保质量

## 明日预告
- 数据分析：检查前4天的曝光和点击数据
```
