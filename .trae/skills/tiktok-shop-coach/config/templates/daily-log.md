# 每日运营日志模板

## 使用说明

此模板用于生成每日运营日志。日志采用**灵活模块化结构**，分为必填模块和可选模块，根据当天实际内容灵活组合。

**参考示例**：
- [6月19日.md](../../../../TikTokShop/2026/6月/6月19日.md) - 标准9模块示例
- [6月20日.md](../../../../TikTokShop/2026/6月/6月20日.md) - 扩展模块示例

---

## 一、必填模块（每次必须包含）

### 模块1：基础信息

```markdown
# {YYYY}年{M}月{D}日 TikTok Shop 运营日志

## 基础信息

### 店铺状态
| 项目 | 情况 |
|------|------|
| 运营天数 | 第{N}天 |
| 店铺类型 | TikTok Shop越南站 |
| 店铺代码 | CNVNCBBRL4FX |
| 类目 | 时尚配件 |
| 运营模式 | 无货源一件代发（1688/拼多多） |
| 预算 | 5000元以内 |
| TikTok账号 | {状态} |

### 成本控制
| 项目 | 情况 |
|------|------|
| 采购成本限制 | 20元以内 |
| 预计运费 | {运费} |
| 目标利润率 | 30%以上 |

### 运营数据
| 指标 | 数值 | 变化 |
|------|------|------|
| 商品数 | {N} | {+/-N 或 不变} |
| 订单数 | {N} | {+/-N 或 不变} |
| GMV | {N} | {+/-N 或 不变} |
| 视频数 | {N} | {+/-N 或 不变} |
| 粉丝数 | {N} | {+/-N 或 不变} |
```

**注意**：如有重要状态变化（如账号注册完成），可单独列出小节：
```markdown
### TikTok账号状态 ✅ 已完成
| 项目 | 情况 |
|------|------|
| TikTok账号 | Cool Base（已注册） |
| 状态 | 已完善资料 |
```

## 二、可选模块（根据当天内容选择）

### A. 问题分析类

#### A1. ⚠️ 重大发现与问题
```markdown
## ⚠️ 重大发现与问题

### 问题{N}：{问题标题}
- **现状**：{当前情况}
- **影响**：{对运营的影响}
- **状态**：{待解决/已解决/搁置}
- **替代方案**：{如有}
```

#### A2. 问题诊断
```markdown
## 问题诊断

{今天遇到的最大问题}

### 分析过程

  - {因素2}
```

```markdown

- **感悟**：{学到了什么}

## 💡 视频素材标记
```

## 四、模块选择指南

| 当天主要内容 | 推荐模块组合 |
|-------------|-------------|
| 日常运营指导 | 基础信息 + 每日复盘 + 问题诊断 + 明日计划 |
| 选品分析 | 基础信息 + 关键决策 + 思考过程 + 成本与利润分析 + 每日复盘 + 明日计划 |
| 定价讨论 | 基础信息 + 定价策略方案 + 定价策略分析 + 每日复盘 + 明日计划 |
| 问题处理 | 基础信息 + ⚠️重大发现与问题 + 问题诊断 + 执行方案 + 每日复盘 + 明日计划 |
| 知识学习 | 基础信息 + 知识学习记录 + 学习要点 + 每日复盘 + 明日计划 |
| 综合运营 | 基础信息 + 所有相关可选模块 + 每日复盘 + 明日计划 + 资源整理 |

## 五、变量说明

| 变量 | 来源 | 说明 |
|------|------|------|
| `{YYYY}` | 当前日期 | 年份 |
| `{M}` | 当前日期 | 月份（数字） |
| `{D}` | 当前日期 | 日期（数字） |
| `{N}` | 计算得出 | 运营天数 = (当前日期 - start_date) + 1 |
| `{状态}` | shop-metrics.json | TikTok账号状态 |
| `{运费}` | shop-metrics.json | 运费信息 |

## 六、写作规范

### 叙事线（面向自媒体）
- 用**第一人称**记录
- 语言**口语化**，适合转为视频脚本
- 包含**情绪表达**（兴奋、沮丧、意外等）
- 突出**冲突和转折**（以为...结果...）
- 提炼**金句**

### 数据线（面向运营）
- 只记录**有变化的指标**
- 标注**变化原因**
- 与前一天数据对比

### 决策记录（面向复盘）
- 列出**所有选项**的优缺点
- 记录**选择原因**和**风险评估**
- 标注**备选方案**
