# 数据分析报告模板

## 使用说明

此模板用于生成数据分析报告。根据用户提供的运营数据，诊断问题并给出优化建议。

## 模板结构

```markdown
# 📊 数据分析报告

## 今日数据
- 曝光：{impressions}
- 点击：{clicks}
- CTR：{ctr}%
- 转化率：{conversion_rate}%
- 订单：{orders}
- GMV：{gmv}

## 问题诊断
- **最大问题**：{问题环节}偏低
- **转化率**：{conversion_rate}%（行业平均{industry_avg}%）

## 优化方案
### 优先级1：{优化点}
- {具体步骤}

### 优先级2：{优化点}
- {具体步骤}

## 预期改善
- CTR：{current_ctr}% → {target_ctr}%
- 订单：{current_orders} → {target_orders}
```

## 变量说明

| 变量 | 来源 | 说明 |
|------|------|------|
| `{impressions}` | 用户输入/shop-metrics.json | 曝光量 |
| `{clicks}` | 用户输入/shop-metrics.json | 点击量 |
| `{ctr}` | 计算得出 | 点击率 = 点击/曝光 × 100% |
| `{conversion_rate}` | 计算得出 | 转化率 = 订单/点击 × 100% |
| `{orders}` | 用户输入/shop-metrics.json | 订单数 |
| `{gmv}` | 用户输入/shop-metrics.json | 成交总额 |

## 计算逻辑

1. **CTR** = 点击 / 曝光 × 100%
2. **转化率** = 订单 / 点击 × 100%
3. **行业平均CTR**：2-3%（东南亚电商）
4. **行业平均转化率**：1-2%（东南亚电商）

## 问题诊断逻辑

| 指标 | 正常范围 | 问题判断 |
|------|----------|----------|
| CTR | 2-3% | <2% 为主图/标题问题 |
| 转化率 | 1-2% | <1% 为价格/详情页问题 |
| 加购率 | 5-10% | <5% 为产品吸引力问题 |

## 示例输出

```markdown
# 📊 数据分析报告

## 今日数据
- 曝光：5,000
- 点击：150
- CTR：3%
- 转化率：1.3%
- 订单：2
- GMV：90,000 VND

## 问题诊断
- **最大问题**：转化率偏低
- **转化率**：1.3%（行业平均1.5%）

## 优化方案
### 优先级1：优化商品详情页
- 增加产品细节图
- 添加使用场景说明
- 优化越南语文案

### 优先级2：调整定价策略
- 检查竞品价格
- 考虑设置优惠券
- 测试不同价格点

## 预期改善
- CTR：3% → 3.5%
- 订单：2 → 3-4
```
