# 选品分析报告模板

## 使用说明

此模板用于生成选品分析报告。根据用户提供的产品信息，进行利润测算和市场分析，给出是否推荐的建议。

## 模板结构

```markdown
# 🔍 选品分析报告：{产品名称}

## 产品信息
- **采购价**：{purchase_price}元
- **建议售价**：{selling_price}元（约{selling_price_vnd} VND）
- **利润率**：{profit_margin}%

## 市场分析
- **市场需求**：{高/中/低}
- **竞争程度**：{红海/蓝海/中等}
- **复购潜力**：{高/中/低}

## 利润测算
| 成本项 | 金额（CNY） | 金额（VND） |
|--------|-------------|-------------|
| 采购价 | {purchase_price} | {purchase_price_vnd} |
| 跨境运费 | {shipping} | {shipping_vnd} |
| 平台佣金 | {commission} | {commission_vnd} |
| 总成本 | {total_cost} | {total_cost_vnd} |
| 利润 | {profit} | {profit_vnd} |

## 综合评分
{评分星级}

## 建议
{推荐/不推荐}
{具体理由}
```

## 变量说明

| 变量 | 来源 | 说明 |
|------|------|------|
| `{产品名称}` | 用户输入 | 待分析产品名称 |
| `{purchase_price}` | 用户输入 | 采购价格（CNY） |
| `{selling_price}` | 计算得出 | 建议售价（CNY） |
| `{selling_price_vnd}` | 汇率换算 | 建议售价（VND） |
| `{profit_margin}` | 计算得出 | 利润率百分比 |
| `{shipping}` | 参数表 | 跨境运费 |
| `{commission}` | 参数表 | 平台佣金（6%） |

## 计算逻辑

1. **总成本** = 采购价 + 跨境运费 + 平台佣金
2. **利润** = 售价 - 总成本
3. **利润率** = 利润 / 售价 × 100%
4. **VND换算** = CNY × 3,872（四舍五入到千位）

## 评分规则

| 利润率 | 竞争度 | 复购潜力 | 评分 |
|--------|--------|----------|------|
| ≥30% | 蓝海 | 高 | ★★★★★ |
| ≥30% | 中等 | 中 | ★★★★☆ |
| 25-30% | 中等 | 中 | ★★★☆☆ |
| <25% | 红海 | 低 | ★★☆☆☆ |

## 示例输出

```markdown
# 🔍 选品分析报告：时尚太阳镜

## 产品信息
- **采购价**：15元
- **建议售价**：45元（约174,240 VND）
- **利润率**：33%

## 市场分析
- **市场需求**：高（夏季刚需）
- **竞争程度**：中等（店铺数约80家）
- **复购潜力**：中（季节性购买）

## 利润测算
| 成本项 | 金额（CNY） | 金额（VND） |
|--------|-------------|-------------|
| 采购价 | 15 | 58,080 |
| 跨境运费 | 3 | 11,616 |
| 平台佣金 | 2.7 | 10,454 |
| 总成本 | 20.7 | 80,150 |
| 利润 | 24.3 | 94,090 |

## 综合评分
★★★★☆

## 建议
**推荐**
理由：利润率达标，市场需求稳定，竞争可控。建议作为夏季主推款。
```
