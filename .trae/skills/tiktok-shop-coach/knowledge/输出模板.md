## 运营教练：每日任务清单

```markdown
# 🎯 今日任务清单（第{N}天）

## 当前状态
- 商品数：{N}
- 订单数：{N}
- 阶段：{冷启动/成长期/稳定期}

## 今日最重要的3件事

### 任务1：{名称}
- **为什么做**：{原因}
- **具体步骤**：
  1. {步骤1}
  2. {步骤2}
  3. {步骤3}
- **预计耗时**：{N}分钟
- **完成标准**：{可验证的标准}
- **验证方法**：{如何确认完成}

### 任务2：{名称}
（同上格式）

### 任务3：{名称}
（同上格式）

## 失败调整
- 任务1失败 → {调整方案}
- 任务2失败 → {调整方案}

## 明日预告
- {预告内容}
```

## 选品专家：选品分析报告

```markdown
# 🔍 选品分析报告：{产品名称}

## 产品信息
- **采购价**：{N}元
- **建议售价**：{N}元（约{N}万VND）
- **利润率**：{N}%

## 市场分析
- **市场需求**：{高/中/低}
- **竞争程度**：{红海/蓝海/中等}
- **复购潜力**：{高/中/低}

## 利润测算
| 成本项 | 金额 |
|--------|------|
| 采购价 | {N}元 |
| 跨境运费 | {N}元 |
| 平台佣金 | {N}元 |
| 总成本 | {N}元 |
| 利润 | {N}元 |

## 综合评分
{★★★★☆}

## 目标人群
{人群画像}

## 视频创意方向
{创意1、创意2}

## 风险点
{风险描述}

## 建议
{推荐/不推荐} + {具体理由}
```
