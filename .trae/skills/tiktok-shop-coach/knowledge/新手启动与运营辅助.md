## 新手启动流程

### 首次使用检测

若 `.trae/data/shop-metrics.json` 不存在或为空，先执行启动问卷收集基础信息：

```
欢迎来到 TikTok Shop 越南站运营助手！为给出精准建议，请先回答：

1. 预算范围（人民币）？ A.5000以内  B.5000-10000  C.10000-30000  D.30000以上
2. 每天可投入时间？     A.1-2小时(兼职)  B.3-4小时(半职)  C.5小时以上(全职)
3. 是否已有稳定货源？   A.有,1688/拼多多  B.有,工厂直供  C.没有,需寻找
4. 是否已有 TikTok 账号？A.有,已有粉丝  B.有,0粉丝  C.没有,需注册

请回复编号+选项，例如：1A 2B 3C 4B
```

收集后写入 `shop-metrics.json` 的 `shop_info`，作为后续建议的基线。

## 风险预警系统

### 账号风险
- 异常粉丝增长（日增 >1000）→ 检查是否刷粉
- 视频播放量突然为 0 → 检查是否限流

### 商品风险
- 退货率 >20% → 立即下架检查
- 差评集中 → 分析问题并优化

### 资金风险
- ROI <1 持续 3 天 → 停止投放广告
- 库存周转 >30 天 → 清仓处理

## 供应商联系模板

**货源模式为一件代发，供应商沟通必须使用中文。**

```
您好！

我是越南 TikTok Shop 的卖家，刚刚在您店里购买了 [商品名称]。

请问：
1. 您是否有产品视频素材可以提供？
2. 是否有更多高清产品图片？
3. 是否支持一件代发到越南？

如果方便，能否通过旺旺/微信发送给我？

谢谢！
```
