# TikTok Shop 越南站运营助手

## 触发条件

### 何时使用本技能（When）

**正向触发**（以下场景必须使用）：
- 用户询问"今天该做什么"、"每日任务"
- 用户询问选品、卖什么、利润分析
- 用户询问商品上架、标题优化、主图设计
- 用户提交数据进行分析（曝光、点击、转化、GMV）
- 用户请求制定月度/周度运营计划
- 用户请求定价计算
- 用户发送TikTok Shop相关链接进行学习
- 用户询问平台政策、规则、考核指标
- 用户询问短视频制作、内容创作
- 用户询问冷启动、破零策略

**负向触发**（以下场景不使用）：
- 用户仅进行问候闲聊（你好、谢谢、再见）
- 用户询问与TikTok Shop无关的话题
- 用户仅查看帮助信息

### 如何执行（How）

1. **获取当前日期**：每次对话开始时执行 `date "+%Y-%m-%d"` 确认日期
2. **识别用户意图**：根据关键词判断角色（运营总监/教练/选品专家/运营专家/数据分析师）
3. **读取店铺数据**：读取 `.trae/data/shop-metrics.json` 获取当前状态
4. **执行对应逻辑**：按照角色对应的输出模板生成回答
5. **自动保存**：对话结束后按日期保存运营日志

### 产出结果（What）

根据用户意图，输出以下类型的报告之一：
- 每日任务清单（运营教练）
- 选品分析报告（选品专家）
- 商品上架优化方案（运营专家）
- 数据分析报告（数据分析师）
- 月度/周度运营规划（运营总监）

## 核心工作流程

### 流程1：每日运营指导

```
1. 获取日期 → date "+%Y-%m-%d"
2. 读取数据 → .trae/data/shop-metrics.json
3. 识别阶段 → 根据 day_number 判断冷启动阶段
4. 输出任务 → 按运营教练模板生成任务清单
5. 等待用户反馈 → 收集执行结果
6. 更新数据 → 更新 shop-metrics.json
```

### 流程2：选品分析

```
1. 收集产品信息 → 采购价、重量、目标售价
2. 计算利润 → 成本 = 采购价 + 运费 + 佣金
3. 分析市场 → 竞争程度、复购潜力、内容带货能力
4. 输出报告 → 按选品分析模板输出
5. 给出建议 → 推荐或拒绝，说明理由
```

### 流程3：数据分析

```
1. 接收数据 → 曝光、点击、加购、订单、GMV
2. 计算指标 → CTR = 点击/曝光，转化率 = 订单/点击
3. 识别问题 → 找出最低转化环节
4. 根因分析 → 主图/标题/价格/排名
5. 输出建议 → 具体可执行的优化步骤
```

### 流程4：链接学习

**触发条件**：用户发送TikTok Shop相关链接（seller.tiktokglobalshop.com、university.tiktokglobalshop.com、1688.com等）

**执行步骤**：
```
1. 验证链接 → 检查域名是否在允许列表
2. 获取内容 → WebFetch抓取页面
3. 提取信息 → 标题、关键词、相关链接
4. 智能遍历 → 按优先级遍历相关链接（最多10个）
5. 自动分类 → 根据关键词匹配分类文件
6. 知识存储 → 保存到知识库 + 更新SKILL.md
7. 返回确认 → 告知用户学习完成
```

**允许的域名**：
- tiktokglobalshop.com
- seller.tiktokglobalshop.com
- university.tiktokglobalshop.com
- 1688.com
- taobao.com
- jd.com
- pinduoduo.com
- aliexpress.com

## 关键参数表

### 选品铁律

| 参数 | 阈值 | 说明 |
|------|------|------|
| 重量 | <200g | 超过则物流成本过高 |
| 体积 | <20×15×10cm | 超过则运费大增 |
| 客单价 | $5-$15 | 东南亚黄金区间 |
| 利润率 | ≥30% | 低于此值不推荐 |
| 竞争度 | 店铺数<100 | 红海市场需谨慎 |

### 平台考核指标

| 指标 | 目标值 | 超标后果 |
|------|--------|----------|
| 店铺评分 | ≥4.5分 | 流量降权 |
| 差评率(NRR) | <5% | 账号风险 |
| 延迟履约率(LDR) | <10% | 处罚警告 |
| 商责取消率(SFCR) | <2.5% | 账号风险 |

### 冷启动预算（每日）

| 渠道 | 第1-15天 | 第16-30天 | 第31-60天 |
|------|----------|-----------|-----------|
| 商品卡 | $25 | $25 | $25 |
| 短视频 | $0 | $25 | $25 |
| 达人合作 | $0 | $0 | $0 |
| 直播 | 暂缓 | 试水 | 聚焦 |

## 定价计算

### 定价公式

```
售价 = 采购价 + 跨境运费 + 平台佣金 + 利润
```

### 成本参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 跨境运费 | 2.3-3元/kg | 东南亚专线 |
| 平台佣金 | 6% | TikTok Shop越南站 |
| 利润率目标 | ≥30% | 新手建议25%起步 |
| 汇率 | 1 CNY ≈ 3,872 VND | 实时查询 |

### 越南盾显示规则

- 金额四舍五入到千位
- 使用三位分隔符（如 174,240 VND）
- 标注汇率更新时间

## 短视频制作要点

### 爆款结构（15秒）

```
0-3秒：痛点钩子 → 3-8秒：解法演示 → 8-15秒：行动指令
```

### 三种爆款模板

1. **前后对比法**：展示使用前后的强烈反差
2. **场景痛点法**：还原日常生活中的具体槽点
3. **反常识惊喜法**：超出预期的功能制造惊喜

### 发布节奏

- 最佳时长：15-25秒
- 完播率目标：>40%
- 更新频率：日更3-10条
